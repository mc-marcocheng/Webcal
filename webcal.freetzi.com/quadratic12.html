如何在程式中加入K字元

　

例題1: 解 x² - 7x + 12 = 0

按 2ndF DEL (必要) 再按 RCL F1 ALGB 1 = 7 +/- = 12 = (顯示判別式為1表示為實數解)

= = (略過兩個中間值(3.5及0.5))

= (顯示第一個實數根為3) = (顯示第二個實數根為4)

∴ x = 3 或 x = 4

注意你亦可以利用中間的兩個數值作為計算的步驟

即: x = 3.5 ± 0.5

∴ x = 4 或 x = 3
