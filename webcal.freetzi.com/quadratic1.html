A C ) + ( +/- B -
√ ( B x² - 4 A C

例題1: 解 x² - 7x + 12 = 0

按 RCL F1 ALGB 1 = 7 +/- = 12 = (顯示第一個解為4) 再按 0 = (顯示另一個解為3)

　

例題2: 解 x² + 6x + 25 = 0

按 RCL F1 ALGB 1 = 6 = 25 = (顯示Error 2，表示無實解)

註: 答案若為小數，可以用內置分數功能嘗試將小數轉換為分數(按a b/c 或 2ndF d/c)，

即使是簡單的分數(例如:1/3)，亦不一定會成功。
