半衰期計算(應用內置指數回歸功能)

編寫日期: 2006年4月4日

例題: 若一放射最初的放射強度為1000 Bq，時間t為20小時的於射強度為125 Bq，求半衰期及時間t為12小時的放射強度。

按 2ndF MODE 3 3 (進入指數回歸模式) 0 (x,y) 1000 DATA 20 (x,y) 125 DATA

再按 500 2ndF x’ (500為最初的半，顯示半衰期6.666666666小時)

再按 12 2ndF y’ (顯示放射強度為287.174887 Bq)

註1: y’ 用於為計算放射強度或未衰變數目，x’則用於計算半衰期。

註2: 若果你已在MODE 3 3的模式，重複計算相似問題，則可按2ndF CA代替再按2ndF MODE 3 3
