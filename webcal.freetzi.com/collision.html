彈性碰撞

若已知兩物體質量及碰撞前的速度，程式可以計算經過彈性碰撞後兩物體的速度。

程式編寫日期: 2006年6月21日

0 A B - C + D +
( ( A C ÷ B + 2
D - C ) ÷ ( 1 +
A ÷ B ) + C - D
) STO F1 或 F2

例題: 物體A的質量及初速度分別為 5 kg及6 ms-1，物體B最初則靜止，質量為 10 kg，若物體A正面撞向物體B，碰揰為完全彈性碰撞，求A及B碰撞後的速度。

先輸入兩物的質量再輸入初速度

按 RCL F1 ALGB 5 = 10 = 6 = 0 = (顯示A的未速度為 - 2 ms-1)

0 = (顯示B的未速度為 4 ms-1)
