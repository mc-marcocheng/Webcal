估數字

程式編寫日期: 2007年3月27日

程式會產生一個 0至999的整數，遊戲的目的是要找出這個數字，每次估數字時，若果估值較大時會顯示1的提示，若果估值較少時，會顯示 - 1的提示，若果估的數字正確，會顯示0。程式要使用兩個算式記憶。

程式第一部份:

1 0 0 0 RANDOM × 0 STO F1

程式第二部份:

A - K ) ÷ √ ( A
- K + 10x +/- 4 9 )
x2 STO F2

如何在程式中加入K字元

　

操作方法:

1. 先按 RCL F1 = 產生一個要估的數值。

2. 再按 RCL F2 ALGB 輸入估計的數值再按 =，這時會顯示 1 或 - 1 或 0 的數值

註: 每次估數字時，若果估值較大時會顯示1的提示，若果估值較少會顯示 - 1的提示，若果估值正確，會顯示0。

3. 若果未估中數字，再按 ALGB 輸入另一個估計的數值 再按 = ，若果未估中數值，請重覆步驟3，直至估中數值(提示為0)。
