例題1: 已知圓內接四邊形的四邊分別為5,12,12及5，求面積。 

按 P1  5 RUN 12 RUN 12 RUN 5 RUN (顯示面積為60)

 

例題2: 己知三角形的三邊分別為3，4及5，求面積。

按 P1  3 RUN 4 RUN 5 RUN 0 RUN (輸入0代表計算三角形面積，顯示面積為6)
