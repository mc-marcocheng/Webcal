複數的整數指數

程式編寫日期: 2006年11月12日

程式(15步)
1 ENT 2. R→P 3. ENT 4. = 5. X←→Y
6. Kin 1 7. X←→Y 8. xy 9. ENT 10. Kin × 1
11. P→R 12. Kout 1 13. = 14. HLT 15. X←→Y

LRN 模式輸入程式(供 fx-3600PV及fx-3800P使用，程式長度: 15步 )
ENT 3 R→P ENT 4 = X←→Y
Kin 1 X←→Y xy ENT 5 Kin × 1
P→R Kout 1 = HLT X←→Y
MODE . 　 　 　 　

　

例題:計算  (2 + 3i )5

按 P1 再按 2 RUN 3 RUN 5 RUN (顯示實數部為 122) RUN (顯示虛數部為 -597 i)

所以 (2 + 3i)5 = 122 - 597i
