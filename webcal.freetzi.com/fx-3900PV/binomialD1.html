二項分佈(I)

程式編寫日期: 2006年10月5日

程式(19步)
1 ENT 2. Kin 1 3. nCr 4. ENT 5. Kin 3
6. ENT 7. Kin 2 8. × 9. 1 10. X←→K3
11. Kin - 3 12. xy 13. Kout 2 14. Kin - 1 15. ×
16. Kout 3 17. xy 18. Kout 1 19. M+ 20. 　

LRN 模式輸入程式(供 fx-3600PV，程式長度: 19步 )
ENT 2 Kin 1 nCr ENT 1 Kin 3
ENT 0 Kin 2 × 1 X←→K3
Kin - 3 xy Kout 2 Kin - 1 ×
Kout 3 xy Kout 1 M+ MODE .

LRN 模式輸入程式(供 fx-3800P，程式長度: 24步 )
ENT 2 Kin 1 x! ÷ ENT 1
Kin 3 ENT 0 Kin 2 Kin - 1 x!
÷ Kout 1 x! × 1
X←→K3 Kin - 3 xy Kout 2 ×
Kout 3 xy Kout 1 M+ MODE .

注意: 程式除了計算答案外，亦會將答案累加在獨立記憶(M)中，若果要計算多個概率的總和，請先清除記憶M的數值(按 AC SHIFT Min)

　

例題1: 若X ~ Bin(9,0.5)，求P(X=4)。

按 P1  9 RUN . 5 RUN 4 RUN (顯示答案為0.24609)

　

例題2: 若X ~ Bin(9,0.5)，求P(X=4)，P(X=7)及P(X=4)+P(X=7)。

按 AC SHIFT Min (必要) P1  9 RUN . 5 RUN 4 RUN (顯示P(X=4)為0.24609)

P1  9 RUN . 5 RUN 7 RUN (顯示P(X=7)為0.07031)

MR (顯示P(X=4)+P(X=7)為0.31641)
