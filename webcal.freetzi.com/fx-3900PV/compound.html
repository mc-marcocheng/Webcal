複利率計算(I)

程式計算複利息(Compound Interest)的問題。只要輸入本利和(Amount)、最初本金(Principal)、年利率(Annual Interest Rate)、存款時間(Deposit Period，以月為單位)，其中三個數值的情況下，程式會根據已知的三個數值，計算第四個數的數值。注意程式需要輸入複利一期結算的時間(以月為單位計算 )。

例題1: 本金為$100 000，年利率4%，存款期為36月，每月複利計息一次。求本利和。

按 P1  再按 RUN (不輸入數值表示計算本利和) 

100 000 RUN (本金)

4 RUN (年利率)

36 RUN (存款時間)

1 RUN (每月計息，顯示本利和為 $112 727.19)
