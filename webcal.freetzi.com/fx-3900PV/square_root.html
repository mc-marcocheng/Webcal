平方根化簡(I)

程式編寫日期: 2006年11月13日

程式可以將一個正整數的平方根(√a)化簡為最簡式(b√c)。程式二較長，但操作較簡單。

程式(18步)
1 Kout 1 2. ÷ 3. 1 4. Kin + 3 5. Kout 3
6. = 7. √ 8. - 9. Fix 0 10. RND
11. Norm 12. Kin 2 13. = 14. x2 15. x > 0
16. Kout 2 17. HLT 18. Kout 3 19. 　 20. 　

LRN 模式輸入程式(只供 fx-3600PV及fx-3800P使用，程式長度: 18步 )

注意輸入程式前請先按 SHIFT KAC
Kout 1 ÷ 1 Kin + 3 Kout 3
= √ - Fix 0 RND
Norm Kin 2 = x2 x > 0
Kout 2 HLT Kout 3 MODE . 　

　

例題: 化簡 √18

按 SHIFT KAC (必要) 18 SHIFT Kin 1  再按 P1 (顯示 3) RUN (顯示 2)

所以 √18 = 3√2
