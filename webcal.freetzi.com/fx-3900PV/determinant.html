例題2: 計算 (i + 2j + 3k) × ( 4i + 5j + 6k)

按 P1  1 RUN 2 RUN 3 RUN 4 RUN 5 RUN 6 RUN

RUN (顯示-3) RUN (顯示6) RUN (顯示-3)

所以(i + 2j + 3k) x ( 4i + 5j + 6k) = -3i + 6j -3k

計算完結後按AC 終止程式。

　

例題3: 計算 (i + 2j + 3k) x ( 4i + 5j + 6k)˙(2i - 3j + 7k)

按 P1 1 RUN  2 RUN 3 RUN 4 RUN 5 RUN 6 RUN  2 RUN 3 +/- RUN 7 RUN (顯示答案為 - 45)

所以(i + 2j + 3k) x ( 4i + 5j + 6k)˙(2i - 3j + 7k) = - 45

計算完結後按AC 終止程式。
