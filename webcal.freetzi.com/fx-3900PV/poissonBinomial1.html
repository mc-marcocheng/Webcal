注意: 程式除了計算答案外，亦會將答案累加在獨立記憶(M)中，若果要計算多個概率的總和，請先清除記憶M的數值(按 AC SHIFT Min)

　

例題1: 若X ~ Bin(9,0.5)，求P(X=4)。

按 0 P1 (0代表計算二項分佈)

9 RUN . 5 RUN 4 RUN (顯示答案為0.24609)

　

例題2: 若X ~ Bin(9,0.5)，求P(X=4)，P(X=7)及P(X=4)+P(X=7)。

按 AC SHIFT Min (必要) 0 P1 (0代表計算二項分佈)

9 RUN . 5 RUN 4 RUN (顯示P(X=4)為0.24609)

0 P1  9 RUN . 5 RUN 7 RUN (顯示P(X=7)為0.07031)

MR (顯示P(X=4)+P(X=7)為0.31641)

　

例題3: 若X ~ Po(5)，求P(X=4)。

 按 1 P1 (1代表計算泊松分佈)

 5 RUN 4 RUN (顯示答案為0.17547)

 

例題4: 若X ~ Po(5)，求P(X=3)，P(X=5)，P(X=6)及P(X=3)+P(X=5)+P(X=6)。

按 AC SHIFT Min (必要) 1 P1 (1代表計算泊松分佈)

 5 RUN 3 RUN (顯示P(X=3)為0.14037)

1 P1  5 RUN 5 RUN (顯示P(X=5)為0.17547)

1 P1  5 RUN 6 RUN (顯示P(X=6)為0.14622)

MR (顯示P(X=3)+P(X=5)+P(X=6)為0.46206)
