例題: 求 110的質因子連乘式。

第一個程式按法:

按 SHIFT KAC (必要) P1  110 RUN (顯示2) RUN (顯示5) RUN (顯示11)

RUN (顯示0表示計算完結)

所以 110 = 2 × 5 × 11

　

第二個程式按法:

按 P1  110 RUN (顯示2) RUN (顯示5) RUN (顯示11)RUN (顯示0表示計算完結)

所以 110 = 2 × 5 × 1
