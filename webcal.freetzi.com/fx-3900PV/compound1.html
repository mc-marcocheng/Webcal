複利率計算(II)

程式用作計算複利率問題，假定利息每月計算及息率固定不變，程式可以計算以下問題:

˙本利和 Amount ( 已知本金、年利率、存款期數及每月存款)

˙本金 Principle (已知本利和、年利率及 存款期數及每月存款)

˙每月存款 Monthly deposit  (已知本利和、本金、年利率及存款期數)

˙存款期數 (已知本利和、本金、年利率 及每月存款)

˙程式亦可以計算簡單複利率問題、零存整付問題及按揭問題

註1: 程式容許指定存款指示的方式，0代表每期結束後存款，1代表每期開始時存款。通常貸款問題選用0，存款問題選用1。

註2: 程式不能計算年利率。

程式編寫日期: 2006年12月1日

例題1: 本金為$7000，年利率為3%，存款期數為12，每月初存款$200，求本利和。

按 P1 再按 RUN (不輸入數值代表計算本利和)

7000 RUN (本金)

3 RUN (年利率)

12 RUN (存款期數)

200 RUN (每月存款)

1 RUN (每月初存款，顯示本利和為 $9652.27)
