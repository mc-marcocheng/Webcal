 例題1: 若X ~ Bin(9,0.5)，求P(4≦X≦6)。

第一個程式的按法:

按 SHIFT KAC (必要) 9 Kin 1  . 5 Kin 2  4 Kin 3  6 SHIFT Min

再按 P1 (顯示答案為0.65625)

第二個程式的按法:

按 P1  再按 9 RUN 0.5 RUN 4 RUN 6 RUN (顯示答案為0.65625)

程式執行完成後，請按AC中止程式。

　

例題2: 若X ~ Bin(9,0.5)，求P(X=4)。

第一個程式的按法:

按 SHIFT KAC (必要) 9 Kin 1  . 5 Kin 2  4 Kin 3 SHIFT Min (記憶相同的值代表只計算一項)

再按 P1 (顯示答案為0.24609)

第二個程式的按法:

按 P1  再按 9 RUN 0.5 RUN 4 RUN RUN (不輸入最後數值代表只計算一項，顯示答案為0.24609)

程式執行完成後，請按AC中止程式。
