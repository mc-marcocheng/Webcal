例題: 計算 (2x4 + x3 + 3x2 + 4x + 5) ÷ (x + 2) 的商及餘數。
　

第一個程式按法:

按 SHIFT KAC (必要) 1 Kin 1 2 SHIFT kin 2 (先輸入除式的係數)

4 SHIFT Min (輸入被除多項式的最高次數)

再按 P1  2 RUN (顯示商 x3 的係數為2) 1 RUN (顯示商 x2 的係數為 - 3)

3 RUN (顯示商 x 的係數為9) 4 RUN (顯示商常數項的係數為 - 14)

5 RUN (顯示餘數為33)

所以商為2x3 - 3x2 + 9x - 14，餘數為33。

計算完結後，按 AC 終止程式。

　

第二個程式按法:

按 P1  再按 1 RUN 2 RUN (先輸入除式的係數)

4 RUN (輸入被除多項式的最高次數)

再按  2 RUN (顯示商 x3 的係數為2) 1 RUN (顯示商 x2 的係數為 - 3)

3 RUN (顯示商 x 的係數為9) 4 RUN (顯示商常數項的係數為 - 14)

5 RUN (顯示餘數為33)

所以商為2x3 - 3x2 + 9x - 14，餘數為33。

計算完結後，按 AC 終止程式。
