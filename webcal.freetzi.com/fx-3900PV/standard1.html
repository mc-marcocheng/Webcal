標準常態分佈概率(I)

這是一個很簡短的程式，程式需要配合內置的積分功能，亦適用於fx-3600PV。

程式編寫日期: 2006年9月23日

程式(11步)
1 x2 2. +/- 3. ex 4. √ 5. ÷
6. 2 7. √ 8. ÷ 9. π 10. √
11. = 12. 　 13. 　 14. 　 15. 　

LRN 模式輸入程式(供 fx-3600PV，程式長度: 11步 )
x2 +/- ex √ ÷
2 √ ÷ π √
= MODE . 　 　 MODE .

　

例題: 計算 P(0≦ Z≦1)，其中 Z ~ N(0, 1)。

程式需要使用內置積分功能

按 MODE 1  P1 再按 0 RUN 1 RUN (顯示P(0≦ Z≦1)為3.41345 × 10-1，亦即是0.341345)

計算完結後，按 MODE 0 返回正常計算模式

　

註1: 計數機的內置積分功能顯示的答案會以有效數字形式顯示，若果想以小數表示，只要再按 = 即可。

註2: 由於程式使用內置積分功能，計算時間可能會較長。
