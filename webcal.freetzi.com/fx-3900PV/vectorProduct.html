按 P1  再按 1 RUN 1 RUN 1 RUN 1 RUN 1 +/- RUN 3 RUN

1 RUN 2 RUN 1 RUN MR (顯示△為 -  2)

6 RUN 5 RUN 10 RUN MR (顯示△x為 - 2)

1 RUN 1 RUN 1 RUN MR (注意顯示 -△y為 4)

1 RUN - 1 RUN 3 RUN MR (顯示△z為 - 6)

所以 x = (- 2)/(- 2) = 1, y = (- 4)/(-2) = 2, z = (- 6)/(- 2) = 3
