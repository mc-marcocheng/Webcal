例題1: 一等差數列，首項為2及公差為3，計算第20項及首20項之和。

按 1 P1 (1代表計算等差數列)

再按 2 RUN 3 RUN 20 RUN (顯示第20項為59) RUN (顯示首20項和為610)

計算完結後按 AC 終止程式

　

例題2: 一等比數列，首項為4及公比為 -3，求第6項及首6項和。

按 0 P1 (0代表計算等比數列)

再按 4 RUN 3 +/- RUN 6 RUN (顯示第6項為 -972) RUN (顯示首6項和為 -728)

　

例題3: 一等比數列，首項為4及公比為0.5，求數列無窮項的和。 

按 0 P1 (0代表計算等比數列)

再按 4 RUN 0.5 RUN (無窮項的總和為8)

計算完結後按 AC 終止程式
