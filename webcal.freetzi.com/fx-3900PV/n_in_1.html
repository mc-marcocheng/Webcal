香港計數機科技協會N合一程式使用方法

編寫日期: 2007年1月11日

　

現在以合併以下兩個程式為例說明N合一程式如何使用

1.  一元二次方程

2. 三角形三邊求面積及周界

　

步驟:

1. 瀏覽香港計數機科技協會的N合一 程式產生網頁

http://hk.geocities.com/hkact/download/Nin1.html

2. 在右邊選擇 Quadratic equation 及再按下 新增(A)→，這時在右邊會出現 Quadratic equation.

3. 在右邊選擇 Area & Perimeter of Triangle by 3 Sides 及按下 新增(A)→，這時在右邊會出現 Area & Perimeter of Triangle by 3 Sides.

4. 按下 建立程式(R)，這時在網頁下方會出現一個 100 步的程式及一個 Mode Table，Mode Table 中的數字代表那個程式，在這個例子中，1代表 Quadratic equation(一元二次方程)，2代表 Area & Perimeter of Triangle by 3 Sides (三角形三邊求面積及周界)。

5. 輸入合併程式

6. 執行程式，假設計算以下問題

問題1: 解 x2 - 7x + 12 = 0

按 P1  RUN RUN RUN (必要，這時會顯示ENT)，

再按 1 RUN (1代表計算一元二次方程)

再按 1 RUN 7 +/- RUN 12 RUN (顯示第一個實數根為4)

RUN (顯示第二個實數根為3) RUN (顯示判別式為 1)

∴ x = 4 或 x = 3

計算完結後 AC 終止程式

　

問題2: 三角形三邊分別為 3, 4 及 5，求三角形面積及周界。

按 P1  RUN RUN RUN (必要，這時會顯示ENT)，

再按 2 RUN (2代表計算三角形面積及周界)

再按 3 RUN 4 RUN 5 RUN (顯示面積為6)

RUN (顯示周界為 12)

計算完結後 AC 終止程式
