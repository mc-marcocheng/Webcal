例題1: 己知四點分別為(0,0)，(3,4)，(-1,7)及(-4,3)，求四邊形面積。

按 P1  0 RUN 0 RUN 3 RUN 4 RUN 1 +/- RUN 7 RUN 4 +/- RUN 3 RUN (顯示面為25)

注意: 四點要依須時針或逆時針方向輸入，否則不能正常計算四邊形面積。

 

例題2: 己知三點分別為(2,3)，(4,5)及(3,7)，求三角形面積。

按 P1  2 RUN 3 RUN 4 RUN 5 RUN 3 RUN 7 RUN 3 RUN 7 RUN (顯示面為3)

注意: 計算三角形面積時，只要將第四點輸入的數值改為第三點的數值即可。
