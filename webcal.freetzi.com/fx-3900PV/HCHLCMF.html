例題: 找48及60的最大公因數及最小倍數及約簡48/60。

第一個程式按法:

按 48 Kin 1 Kin 2 60 Kin 3 Kin 4 再按 P1 (顯示12) RUN (顯示4) RUN (顯示5) RUN (顯示240)

因此，最大公因數為 12，最小公倍數為240及約簡分數為 4/5。

第二個程式按法:

按 P1  48 RUN 60 RUN (顯示12) RUN (顯示4) RUN (顯示5) RUN (顯示240)

因此，最大公因數為 12，最小公倍數為240及約簡分數為 4/5。

　

註: 輸入的兩數必須為正數。
