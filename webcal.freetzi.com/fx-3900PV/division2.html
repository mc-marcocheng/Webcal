例題: 計算 (3x3 - 5x2 -  4x + 5) ÷ (x2 - x - 1)的商及餘數。

 

第一個程式按法:

按 3 Kin 1 5 +/- Kin 2 4 +/- Kin 3 5 Kin 4 1 Kin 5 1 +/- Kin 6 1 +/- SHIFT Min

再按 P1 (顯示商的x係數為 3) RUN (顯示商的常數項為 -2)

RUN (顯示餘式的x係數為 -3) RUN (顯示餘式的常數項為3 )

　

按 P1  3 RUN 5 +/- RUN 4 +/- RUN 5 RUN

1 RUN 1 +/- RUN 1 +/- RUN (顯示商的x係數為 3) RUN (顯示商的常數項為 -2)

RUN (顯示餘式的x係數為 -3) RUN (顯示餘式的常數項為3 )

所以商為3x - 2，餘式為 -3x + 3。

　

註: 程式執行完結後，按 Kout 1及Kout 2會顯示商式的係數，按 Kout 3及Kout 4 顯示餘式的係數。
