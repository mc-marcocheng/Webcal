註1: 執行程式，輸入扇形變數的次序為半徑、中心角、弧長、面積，注意只可以輸入兩個已知變數，其餘的兩個變數不輸入數值(必要)，即直接按RUN即可。

註2: 注意本程式的中心角以度數為單位。

註3: 程式不能計算變數為0或負數的情況。

　

例題1: 扇形的半徑為25 cm，中心角為216°，求弧長及面積。

按 P1 再按 25 RUN 216 RUN

RUN RUN (不輸入弧長及面積，顯示弧長為94.248 cm)

RUN (顯示面積為 1178.10 cm²)

　

例題2: 扇形的半徑為25 cm，弧長為94.2478 cm，求中心角及面積。

按 P1 再按 25 RUN RUN (不輸入中心角) 94.2478 RUN

RUN (不輸入面積，顯示中心角為216°)

RUN (顯示面積為 1178.10 cm²)

　

例題3: 扇形的半徑為25 cm，面積為1178.10 cm²，求中心角及弧長。

按 P1 再按 25 RUN RUN RUN (不輸入中心角及弧長)

1178.1 RUN (顯示中心角為216°)

RUN (顯示弧長為 94.248 cm)

計算完結後按 AC 終止程式

　

例題4: 扇形的中心角為216°，弧長為 94.2478 cm，求半徑及面積。

按 P1 再按 RUN (不輸入半徑) 216 RUN 94.2478 RUN

RUN (不輸入面積，顯示半徑為25 cm)

RUN (顯示面積為 1178.10 cm²)

　

例題5: 扇形的中心角為216°，面積為 1178.10 cm²，求半徑及弧長。

按 P1 再按 RUN (不輸入半徑) 216 RUN RUN (不輸入弧長)

1178.1 RUN (顯示半徑為25 cm)

RUN (顯示弧長為 94.2479 cm)

計算完結後按 AC 終止程式

　

例題6: 扇形的弧長為94.2478 cm，面積為 1178.10 cm²，求半徑及中心角。

RUN (顯示中心角為 216°)

計算完結後按 AC 終止程式
