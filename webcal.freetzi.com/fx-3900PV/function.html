函數計算

編寫日期: 2006年11月1日

透過編寫簡單程式可以重複計算函數值，編寫方法很簡單，要注意程式的第一步必為 Min，接著好似一般計算，計算函數值即可以下例子可以作為參考。

　

例題: 若 f(x) = x3 + 3x - 2，求 f(0), f(1), f(2)及f(3)

EDIT模式輸入程式

按 MODE 0 P1
1 Min 2. xy 3. 3 4. + 5. 3
6. × 7. MR 8. - 9. 2 10. =

輸入完結後按 MODE .

LRN 模式輸入程式

按 0 MODE EXP P1
Min xy 3 + 3
× MR - 2 =
MODE . 　 　 　 　

　

按 0 P1(顯示f(0)為 - 2) 1 P1 (顯示f(1)為2)

2 P1 (顯示f(2)為12) 3 P1 (顯示f(3)為34)
