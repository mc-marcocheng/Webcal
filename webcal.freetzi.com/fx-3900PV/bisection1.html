例題: 解 x3 – 2x – 1 = 0, 1 < x < 2。

假設程式第一部份及第二部份分別儲存於P1及P2

按 1 P2 (顯示 f(1)為 -2) 2 P2 (顯示 f(2)為 3)

再按 P1 (注意啟動程式時數值不可以是0) 1 RUN (先輸入函數值為負數的x=1)

2 RUN (顯示第一個近似值1.5)  P2 (顯示 f(1.5)的值為 -0.625)

P1 (顯示第二近似值1.75) P2 (顯示 f(1.75)的值為 0.859375)
