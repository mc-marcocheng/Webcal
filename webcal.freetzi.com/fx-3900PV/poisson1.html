泊松分佈(I)

程式編寫日期: 2006年10月2日

程式(12步)
1 ENT 2. xy 3. Kin 1 4. ENT 5. Kin 2
6. ÷ 7. Kout 1 8. ex 9. ÷ 10. Kout 2
11. x! 12. M+ 13. 　 14. 　 15. 　

LRN 模式輸入程式(供 fx-3600PV及fx-3800P使用，程式長度: 12步 )
ENT xy Kin 1 ENT Kin 2
÷ Kout 1 ex ÷ Kout 2
x! M+ MODE . 　 　

注意: 程式除了計算答案外，亦會將答案累加在獨立記憶(M)中，若果要計算多個概率的總和，請先清除記憶M的數值(按 AC SHIFT Min)

　

例題1: 若X ~ Po(5)，求P(X=4)。

 按 P1  5 RUN 4 RUN (顯示答案為0.17547)

 

例題2: 若X ~ Po(5)，求P(X=3)，P(X=5)，P(X=6)及P(X=3)+P(X=5)+P(X=6)。

按 AC SHIFT Min (必要) P1  5 RUN 3 RUN (顯示P(X=3)為0.14037)

P1  5 RUN 5 RUN (顯示P(X=5)為0.17547)

P1  5 RUN 6 RUN (顯示P(X=6)為0.14622)

MR (顯示P(X=3)+P(X=5)+P(X=6)為0.46206)
