例題1: 一等比數列，首項為4及公比為 -3，求第6項及首6項和。

按 P1  4 RUN 3 +/- RUN 6 RUN (顯示第6項為 -972) RUN (顯示首6項和為 -728)

　

例題2: 一等比數列，首項為4及公比為0.5，求數列無窮項的和。 

按 P1  4 RUN 0.5 RUN (無窮項的總和為8)

計算完結後按 AC 終止程式

　

註: 程式執行完結後，按 Kout 1及Kout 2分別顯示N項及N項和的值。
