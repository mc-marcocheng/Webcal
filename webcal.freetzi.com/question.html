解答:

1. 輸入程式有何步驟 ?

答: 輸入程式的步驟很簡單，首先按出所需要的算式，再按 STO F1 或 F2 儲存算式。當要再次使用時，按 RCL F1 或 F2 提取算式即可。

　

2. 輸入程式時，一些英文字元(例如: A)不知如何輸入？

答: 要按出英文字A、B、C、D、X、Y及M，只要按 2ndF ALPHA 再按有那個英文字的按鈕即可，例如: 要按出A字，按 2ndF ALPHA  A (亦即是按 2ndF  RCL  CNST)。

　

要按的特別的K字元，可以按照以下步驟:

1. 先按ON/C 0 + 0 = 0 = 答案除了顯示0外，還有顯示算式0 + K。

2. 按 à DEL DEL 現在算式只餘下K。

3. 按 STO F1 (或 F2) 現在K字元已儲存在F1 (或 F2)。

4. 編寫含K字元的程式，當需要K字元時，按RCL F1 (或 F2)。

5. 編寫完程式後，按STO F1 (或 F2) 儲存程式。

　

總結，產生K及將K儲存於F1或F2的按鍵如下:

0 + 0 = 0 = à DEL DEL STO F1 (或F2)

若要取回K，按 RCL F1 (或F2)

　

3. 如何計算次方(例如: 210)及開多次方(例如: 5√32)的問題？

答: 使用計數機上 yx 及 x√   的功能，

例如:計算210 

按 2  yx  10 =  (顯示算式 2^10= 及答案1024)

計算5√32

按 5  2ndF x√  32 (顯示算式 5x√32= 及答案 2)

　

4. 小數位及有效數字是如何設定及取消？

答: 不斷按 2ndF FSE (亦即是按 2ndF . ) 直至想要的顯示模式。小數顯示形式會有 FIX 的細字，有效數字顯示形式會 有 SCI 的細字，工程顯示形式會有 ENG 的細字，若果沒有指定以上三種顯示形式，計數機會沒有 FIX、SCI及ENG的細字。

若果已切換至 FIX、SCI或ENG的其中一種模式，再按 2ndF TAB (亦即是按 2ndF +/-) 再按數字按鈕(0-9)，可以設定顯示小數點後的位數。

　

5. 計數的操作模式如何設定?

答: 計數的顯示屏對上印有不同模式的按法。

正常計模式: 按 2ndF MODE 0 (無特別的細子顯示)

複數模式: 按 2ndF MODE 1 (有xy或 rθ 的細子顯示)

聯立三元一次方程模式: 按 2ndF MODE 2 (無特別的細字顯示，但會有方程係數要求的輸入符號，例如: a1?)

統計模式: 按 2ndF MODE 3 再按所要的統計模式編號，計數機的PLAYBACK與顯示屏間印有不同統計模式的編號

普通統計模式(SD):  0

線性回歸模式(a + bx): 1

二次回歸模式(... + cx2):  2

指數回歸模式(ex): 3

對數回歸模式(ln x): 4

冪次回歸模式(axb): 5

倒數回歸模式(1/x): 6

例如要進入普通統計模式，按 2ndF MODE 3 0

　

6. 三角函數的度數及弧度是如何設定？

答: 不斷按 2ndF DRG (亦即是按 2ndF 8 )，可切換角度單位，度單位時計數機會顯示 DEG 細字，弧度單位會顯示 RAD 細字，百分度單位會顯示 GRAD 細字。

　

7. 如何將計算的結果儲存在計數機中？

答: 可以儲存數值的字母記憶有A、B、C、D、X、Y及M，例如要將數值儲存到記憶A，只要按 STO A (亦即是按 STO CNST)，提取記憶A的數按 RCL A，要按出記憶A的字母按 2ndF ALPHA A (亦即是按 2ndF RCL CNST)。

　

8. 為何計算平均數及標準差總是很容易出錯？

答: 出現錯誤的原因可能很多，可以注意下列幾點，看看情況可否改善。

(1) 在輸入新數據前，必定要先清除原先的統計記憶 (按 2ndF CA)，否則計算很可能出錯，不過按 MODE 3 0除了切換操作模式外，同時亦會清除原先的統計記憶。

(2) 由於可能要輸入不少的統計數據，輸入時要較為小心，輸入完成後計數機會同時顯示輸入數據的總數，請核對真實數據的數目是否相同，若果不相同表示輸入錯誤。

　

9. 如何輸入有10的指數部分的數值(例如: 2.1×1011)？

答: 使用 Exp 這個按鍵，例如: 2.1×1011 按 2.1 Exp 11

　

10. EL-506V使用何類電池，如何更換電池？

答: EL-506V使用的電池為 LR44 × 2。

更換電池的步驟如下:

(1) 按 2ndF OFF 關閉電源。

(2) 將計數機的背面上方的2個螺絲擰開。

(3) 推開電池蓋。

(4) 用尖頭工具撬出舊電池。

(5) 把新電池的正極 + 向上地裝入機內。

(6) 蓋上電池蓋並擰上螺絲。

(7) 按 RESET 鍵。

　

11. EL-506V可否單靠光能運作？

答: 一般室內的燈光下，EL-506V是不能正常運作，除非光線很充足的情況下(例如:在陽光之下)，因此若果發覺計數機的字體變得暗淡，請立即更換新電池。
