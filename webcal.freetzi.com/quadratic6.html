一元二次方程式(VI)

若果輸入的三個係數為整數，而且根為有理數時，答案將會以分數顯示。　

1 a b/c 2 ÷ A × 2 √
( B x² - 4 A C )
+ 1 a b/c 2 ÷ A × (
+/- B - √ ( B x² -
4 A C ) ) STO F1 或 F2

例題1: 解 7x² + 8x + 1 = 0

按 RCL F1 ALGB 7 = 8 = 1 = (顯示第一個解為-1/7) 再按 0 = (顯示另一個解為-1)

　

例題2: 解 x² + 6x + 25 = 0

按 RCL F1 ALGB 1 = 6 = 25 = (顯示Error 2，表示無實解)

　

註: 答案可能會出現帶分數，若要顯示假分數答案，可以再按2ndF d/c。
