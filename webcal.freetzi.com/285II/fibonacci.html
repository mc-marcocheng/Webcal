斐波那契數列

程式 (共35字，藍色的 - 是負號):

( ( ( 1 + √( 5 ) ) / 2 ) ^ A - ( ( 1 + √( 5 ) ) / - 2 ) ^ - A ) / √( 5 )

SAVE PROG1 或 PROG2 (SRP-285II計數機)

STO EQN (HP 30S計數機)

　

例題: 求斐波那契數列的第十項(F10)。

SRP-285II 按法:  

按 P/V RCL，再按 ← 選擇正確的PROG，再按 = =  10 = (顯示答案為55)

　

HP 30S 按法:

按  VRCL，再按 ← = = 10 = (顯示答案為55)

　

注意事項: 在輸入每一個數據前請先按CL清除之前數據，否則可能出錯，不過若果你輸入的數據的長度等於或長於原先數據的長度，亦可以不按CL。另一個方法是在執行程式之前先按 2nd CL-VAR清除所有數據。
