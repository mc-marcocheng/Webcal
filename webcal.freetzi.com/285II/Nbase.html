基數運算

按鍵說明:

MODE 2 : 設定為基數計算模式 (在計數顯示屏右方會顯示現時設定在那種基數)。

DMS : 基數功能選單 (BASE  d  h  b  o)

         BASE 選單: 代表要設定現時是那種基數

                          D : 設定現時為十進制

                          H : 設定現時為十六進制

                          B : 設定現時為二進制

                          O : 設定現時為八進制

         d : 代表那個數是十進制

         h : 代表那個數是十六進制

         b : 代表那個數是二進制

         o : 代表那個數是八進制

　

注意以下例題都是在基數模式中進行 (按 MODE 2 進入基數模式)

完成所有基數計算後，按 MODE 0 返回正常計算模式。

　

例題1: 將2610轉為二進制、八進制及十六進制。

按 DMS 並按 --> 或 <-- 選"d" 然後按數字 26 =

(注意若果現時是十進制，可以不按DMS 並選"d"，直接按26 = )

這時會顯示當時設定那種進制的數值。

再按 DMS 並選 "BASE" 然後用--> 或 <--選 "H" (顯示1A h，即是1A16)，

再選 "B" (顯示11010 b，即是110102)，

再選 "O" (顯示32 o，即是328)

　

例題2: 計算 1FA16 + 5078，最後答案以十六進制表示。

先設定現時為十六進制，按 DMS ，選 BASE 再選 H

(注意若果現時為十六進制，則可以不按)

再按 1FA + DMS 再選 o 507 = (顯示341 h，即是34116)

　

例題3: 計算 10112 AND 11002

先設定現時為二進制，按 DMS ，選 BASE 再選 B

(注意若果現時為二進制，則可以不按)

再按 1011 AND 1100 = (顯示1000b，即是10002)
