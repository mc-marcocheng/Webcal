例題: 解 x3 – 2x – 1 = 0, 1 < x < 2。

先計算f(1)= -2<0及f(2)=3>0

SRP-285II 按法: 

按 P/V RCL，再按 ← 選擇正確的PROG，再按 =  輸入函數方程，變數要用Ans，再按 =

1 = (先輸入函數值為負數的x=1) 2 =  0 = (0表示第一近值，顯示1.5)

↑1 = (顯示第二近似值1.75)

↑2 = (顯示第三近似值1.625)

↑3 = …………(直至要求的凖確度)

 

HP 30S 按法:

按 VRCL，再按 ← = 再 輸入函數方程，變數要用Ans，再按 =

1 = (先輸入函數值為負數的x=1) 2 =  0 = (0表示第一近值，顯示1.5)

↑1 = (顯示第二近似值1.75)

↑2 = (顯示第三近似值1.625)

↑3 = …………(直至要求的凖確度)

　

注意事項: 在輸入每一個數據前請先按CL清除之前數據，否則可能出錯，不過若果你輸入的數據的長度等於或長於原先數據的長度，亦可以不按CL。另一個方法是在執行程式之前先按 2nd CL-VAR清除所有數據。
