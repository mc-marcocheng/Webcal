幾何分佈 (Geometric distribution)

程式 (共19字):

( 1 - A ) ^ ( B - 1 ) - ( 1 - A ) ^ C

SAVE PROG1 或 PROG2 (SRP-285II計數機)

STO EQN (HP 30S計數機)

　

例題1: 若 X ~Ge(0.5)，求P(X=4)。

SRP-285II 按法:  

按 P/V RCL，再按 ← 選擇正確的PROG，再按 = =

0.5 =  4 = 4 = (第二及第三個輸入數值相同代表只計算一項，顯示答案為0.0625)

 

HP 30S 按法:

按 VRCL，再按 ← = = 0.5 =  4 = 4 = (第二及第三個輸入數值相同代表只計算一項，顯示答案為0.0625)

　

例題2: 若X ~ Ge(0.5)，求P(4≦X≦6)。

SRP-285II 按法:  

按 P/V RCL，再按 ← 選擇正確的PROG，再按 = =

0.5 =  4 = 6 = (顯示P(4≦X≦6)為0.109375)

 

HP 30S 按法:

按 VRCL，再按 ← = = 0.5 =  4 = 6 = (顯示P(4≦X≦6)為0.109375)

　

　

注意事項: 在輸入每一個數據前請先按CL清除之前數據，否則可能出錯，不過若果你輸入的數據的長度等於或長於原先數據的長度，亦可以不按CL。另一個方法是在執行程式之前先按 2nd CL-VAR清除所有數據。
