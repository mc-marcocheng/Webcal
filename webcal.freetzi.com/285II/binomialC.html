正整數冪二項式係數

程式編寫日期: 2008年4月7日

程式: (共55字，注意˙按乘號 ×)

()ABCM + X nCr ( (BX - Y) / (B - M) ) ˙C^( (BX - Y) / (B - M) ) A^( X - (BX - Y) / (B - M

SAVE PROG1 或 PROG2 (SRP-285II計數機)

STO EQN (HP 30S計數機)

　

註1: 二項式形式為 (axb + cxd)n，順序輸入a, b, c, d, n及要計算變數冪數。

註2: 若有方根形式，冪數以分數形式輸入，例如: 平方根以 1/2 輸入，若果是分數變數，請以負冪數輸入，若要計算常數項，請最後冪數輸入0)

　

例題1: 二項式(x2 - 3)9 展開後，求x6的係數。

按 P/V RCL，再按 ← 選擇正確的PROG，再按 = =  

1 = 2 = -3 = 0 = (0代表常數項)

9 = 6 = (顯示x6係數為61236)

　

例題2: 二項式(2x2 - 3/x )9 展開後，求常數項的係數。

　

按 P/V RCL，再按 ← 選擇正確的PROG，再按 = =  

2 = 2 = -3 = -1 = (負冪數代表分數)

9 = 0 = (0冪數代表計算常數項，顯示常數項為489888)

　

注意事項: 在輸入每一個數據前請先按CL清除之前數據，否則可能出錯，不過若果你輸入的數據的長度等於或長於原先數據的長度，亦可以不按CL。另一個方法是在執行程式之前先按 2nd CL-VAR清除所有數據。
