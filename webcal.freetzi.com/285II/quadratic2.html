判別式及一元二次方程

這個程式可計別判式的值，並可以計複數解。

程式 (共70字，藍色的 - 是負號，按EXP或E可以產生E。):

2-1 A-1 ( √( √( ( B2 - 4AC )2 ) ) ( - 1 ) ^ X ( X2 - X ) ^ E - 99 - B ( X2 - 2X )2 ^ E - 99 )

+ ( B2 - 4AC ) cos( sin-1( X ^ E - 99 ) )

SAVE PROG1 或 PROG2 (SRP-285II計數機)

STO EQN (HP 30S計數機)

　

例題1: 解 x2 - 7x + 12 = 0

SRP-285II 按法:  

按 P/V RCL，再按 ← 選擇正確的PROG，再按 = =  

1 = - 7 = 12 = 0 = (0表示判別式，答案為1表示為實數解)

↑3 = (3及4表示計實數解，顯示第一個實數根為4)↑4 = (顯示第二個實數根為3)

∴ x = 4 或 x = 3

 

HP 30S 按法:

按  VRCL，再按 ← = = 1 = - 7 = 12 = 0 = (0表示判別式，答案為1表示為實數解)

↑3 = (3及4表示計實數解，顯示第一個實數根為4) ↑4 = (顯示第二個實數根為3)

∴ x = 4 或 x = 3

　

例題2: 解 x2 + 6x + 25 = 0

SRP-285II 按法:  

按 P/V RCL，再按 ← 選擇正確的PROG，再按 = =  

1 = 6 = 25 = 0 = (0表示判別式，答案為-64表示為 複數解)

↑1 = (1及2表示計複數解，顯示實數部為-3) ↑2 = (顯示虛數部為4)

所以方程的解為 x = -3 ± 4i

　

HP 30S 按法:

按  VRCL，再按 ← = = 1 = 6 = 25 = 0 = (0表示判別式，答案為-64表示為 複數解)

↑1 = (1及2表示計複數解，顯示實數部為-3) ↑2 = (顯示虛數部為4)

所以方程的解為 x = -3 ± 4i

　

注意事項: 在輸入每一個數據前請先按CL清除之前數據，否則可能出錯，不過若果你輸入的數據的長度等於或長於原先數據的長度，亦可以不按CL。另一個方法是在執行程式之前先按 2nd CL-VAR清除所有數據。
