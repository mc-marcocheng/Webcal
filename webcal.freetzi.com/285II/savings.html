零存整付計算

程式編寫日期: 2006年2月9日

程式 (共62字):

12 XA( 1 + B/12) ( ( 1 + XB/12 ) ^ C - 1 )/B +

log( CB/12A ( 1 + B/12) + 1 ) cos( sin-1( X ) ) / log( 1 + B/12)

SAVE PROG1 或 PROG2 (SRP-285II計數機)

STO EQN (HP 30S計數機)

　

例題1: 每月存款3000元，年利率為4%，預計存款24個月，求本利和。

SRP-285II 按法:  

按  P/V RCL，再按 ← 選擇正確的PROG，再按 = =

1 = (1代表計算本利和) 3000 = 0.04 = 24 = (顯示本利和為75078.09)

HP 30S 按法:

按  VRCL，再按 ← = =

1 = (1代表計算本利和) 3000 = 0.04 = 24 = (顯示本利和為75078.09)

　

例題2: 每月存款3000元，年利率為4%，預計本利和為$75078，求存款期數。

SRP-285II 按法:  

按  P/V RCL，再按 ← 選擇正確的PROG，再按 = =

0 = (0代表計算存款期數) 3000 = 0.04 = 75078 = (顯示存款期數為24)

HP 30S 按法:

按  VRCL，再按 ← = =

0 = (0代表計算存款期數) 3000 = 0.04 = 75078 = (顯示存款期數為24)

　

注意事項: 在輸入每一個數據前請先按CL清除之前數據，否則可能出錯，不過若果你輸入的數據的長度等於或長於原先數據的長度，亦可以不按CL。另一個方法是在執行程式之前先按 2nd CL-VAR清除所有數據。
