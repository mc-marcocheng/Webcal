製程能力指數

編寫日期: 2006年4月3日

例題: 若數據為 11, 12, 13, 16, 18，USL = 17 及 LSL =12，

求製程準確度Ca、製程精確度Cp及製程能力Cpk。

　

按 MODE 1 再按 → 或 ← 選擇普通統計 1-VAR =

計算前先清除統計記憶，按 MODE 1 再按 ← 選擇 D-CL =

再按 DATA = 11 ↓↓12↓↓13↓↓16↓↓18↓↓

再按 DATA DATA → 選擇 LIMIT-SET = 17↓12↓

再按  STATVAR 再不斷按→選擇 Cax (顯示Ca為 0.2)

→ (顯示Cp為0.319569) → (顯示Cpk為0.255655)

完成計算後按 MODE 0 返回正常計算模式
