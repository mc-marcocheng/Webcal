三邊求三角形面積(II)

這個程式應用了三角函數的方法計算三角形的面積，當然答案與希羅公式一樣。

程式 (共22字):

AB sin( cos-1( ( A2 + B2 - C2 ) / 2AB ) ) / 2

SAVE PROG1 或 PROG2 (SRP-285II計數機)

STO EQN (HP 30S計數機)

　

例題: 己知三角形的三邊分別為3，4及5，求面積。

SRP-285II 按法:

按 P/V RCL，再按 ← 選擇正確的PROG，再按 = =

3 = 4 = 5 = (顯示答案為6)

　

HP 30S 按法:

按 VRCL，再按 ← = = 3 = 4 = 5 = (顯示答案為6)

　

注意事項: 在輸入每一個數據前請先按CL清除之前數據，否則可能出錯，不過若果你輸入的數據的長度等於或長於原先數據的長度，亦可以不按CL。另一個方法是在執行程式之前先按 2nd CL-VAR清除所有數據。
