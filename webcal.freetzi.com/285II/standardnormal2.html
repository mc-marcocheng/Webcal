標準常態分佈(II)

編寫日期: 2005年12月20日

SRP-285II其實有內置概率計算功能，不過要輸入統計數據後才可以使用，要有標準常態分佈概率計算功能，要先輸入 -1 及 1這兩個數據，使平均數及標準差分別為 0 及 1。

　

程式設定: (注意: - 是按負值鍵)

按 Mode 1再按 → 或 ← 選擇清除統計記憶 D-CL =

按 Mode 1 再按 → 或 ← 選擇普通統計 1-VAR =

按 DATA = - 1 ↓↓ 1 ↓↓ DATA

現在程式己輸入完畢，計數機已有內置標準常態分佈概率的計算功能，程式可以一直保留(包括關機，不過若果改變統計數據，程式將會無效。

輸入完成後可以按Mode 0返回正常計算模式

　

例題: 計算 P(Z≧1)、 P(0≦ Z≦1)及 P(Z≦1)，其中 Z ~ N(0, 1)。

按 MODE 1 再按 → 或 ← 選擇普通統計 1-VAR =

按 DATA ← = 1 = STATVAR ← (顯示R(t)為0.1587)

再按 ← (顯示Q(t)為0.3413)

再按 ← (顯示P(t)為0.8413)

計算完成後按 Mode 0返回正常計算模式。

所以:

P(Z≧1)=0.1587

P(0≦ Z≦1)=0.3413

P(Z≦1)=0.8413
