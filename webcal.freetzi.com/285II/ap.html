等差數列

程式 (共27字):

( A + BC - B ) sin( cos-1( X ) ) + ( 2A + BC - B ) CX / 2

SAVE PROG1 或 PROG2 (SRP-285II計數機)

STO EQN (HP 30S計數機)

　

例題: 一等差數列，首項為2及公差為3，計算首20項之和及第20項。 

SRP-285II 按法:

按 P/V RCL，再按 ← 選擇正確的PROG，再按 = =

2 = 3 = 20 =  0 = (0表示計項值，顯示第20項為59)

↑1= (1表示計項的和，顯示首20項和為610)

　

HP 30S 按法:

按 VRCL，再按 ← = = 2 = 3 = 20 =  0 = (0表示計項值，顯示第20項為59)

↑1= (1表示計項的和，顯示首20項和為610)

　

注意事項: 在輸入每一個數據前請先按CL清除之前數據，否則可能出錯，不過若果你輸入的數據的長度等於或長於原先數據的長度，亦可以不按CL。另一個方法是在執行程式之前先按 2nd CL-VAR清除所有數據。
