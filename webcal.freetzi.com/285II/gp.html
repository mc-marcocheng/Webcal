等比數列

程式 (共35字，藍色的 - 是負號，按EXP或E可以產生E。):

AB ^ ( C - 1 ) sin( cos-1( X ) ) + A ( 1 - B ^ CC ^ E - 99 ) ( 1 - B )-1 X

SAVE PROG1 或 PROG2 (SRP-285II計數機)

STO EQN (HP 30S計數機)

　

例題一: 一等比數列，首項為4及公比為 -3，求首6項和及第6項。

SRP-285II 按法: 

按 P/V RCL，再按 ← 選擇正確的PROG，再按 = =

4 = - 3 =  6 = 0 = (0代表計項值，顯示第6項為 -972)

↑1 = (1表示計項的和，顯示首6項和為 -728)

　

HP 30S 按法:

按 VRCL，再按 ← = = 4 = - 3 =  6 = 0 = (0代表計項值，顯示第6項為 -972)

↑1 = (1表示計項的和，顯示首6項和為 -728)

　

例題二: 一等比數列，首項為4及公比為0.5，求數列無窮項的和。

SRP-285II 按法:

按 P/V RCL，再按 ← 選擇正確的PROG，再按 = =

4 = 0.5 = 0 = 1 = (0代表為無窮項，1代表計項的和，顯示答案為8)

　

HP 30S 按法:

按 VRCL，再按 ← = = 4 = 0.5 = 0 = 1 = (0代表為無窮項，1代表計項的和，顯示答案為8)

　

注意事項: 在輸入每一個數據前請先按CL清除之前數據，否則可能出錯，不過若果你輸入的數據的長度等於或長於原先數據的長度，亦可以不按CL。另一個方法是在執行程式之前先按 2nd CL-VAR清除所有數據。
