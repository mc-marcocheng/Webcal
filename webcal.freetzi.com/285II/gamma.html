伽瑪函數

　

程式編寫日期: 2005年11月19日

　

這個程式利用了 Stirling formula 去計算伽瑪函數的數值，不過對於細變數(<5)，所得的結果的誤差會較大。

 

程式 (共63字)

　

e^( X ln( X ) - X + ln( √( 2π/X ) ) + ( ( ( ( 1/99X2 - 1/140 ) / X2 +

　

1/105 ) / X2 - 1/30 ) / X2 + 1 ) / 12X

SAVE PROG1 或 PROG2 (SRP-285II計數機)

STO EQN (HP 30S計數機)

　

　

例題: 計算 Γ(7.5) 的值。

　

SRP-285II 按法:   

按 P/V RCL，再按 ← 選擇正確的PROG，再按 = =  7.5 = (顯示答案為1871.25)

　

HP 30S 按法:

按  VRCL，再按 ← = = 7.5 = (顯示答案為1871.25)

　

注意事項: 在輸入每一個數據前請先按CL清除之前數據，否則可能出錯，不過若果你輸入的數據的長度等於或長於原先數據的長度，亦可以不按CL。另一個方法是在執行程式之前先按 2nd CL-VAR清除所有數據。
