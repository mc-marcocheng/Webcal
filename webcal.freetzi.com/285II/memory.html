記憶計算

編寫日期: 2007年1月18日

　

記憶的操作

將數值儲存到獨立記憶:  按 SHIFT STO 再按 記憶代號(銀白色的英文字A, B, C, D, E, F, M) 

將記憶的數值取出: 按 RCL 再按 記憶代號

例如: 將 99 儲存在 C 記憶

按 SHIFT STO C (亦即是按 SHIFT RCL hyp)

要取出這個數值按 RCL C (亦即是按 RCL hyp，這時會顯示99)

獨立記憶 M

增加獨立記憶的數值:  按 M+ (注意M+是一個按鍵)

例如: 按 30 M+

減少獨立記憶的數值:  按 SHIFT M- (亦即是按 SHIFT M+)

例如: 按 50 SHIFT M-

顯示及提取獨立記憶:  按 MR 或 RCL M

註: M+及M-還有相當於等號=的完結功能。

完結指令

注意記憶的操作 STO 或 M+ 或 M- 為完結指令，相當於等號 = 的完結功能。

例如: 將 2 + 3 的答案儲存在記憶 C

可以按 2 + 3 SHIFT STO C 因此可以不用按 等號 = 。
