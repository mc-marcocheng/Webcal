正整數冪二項定理

程式 (共18字):

A ^ ( 0B + C - X ) B ^ X ( C nCr X)

SAVE PROG1 或 PROG2 (SRP-285II計數機)

STO EQN (HP 30S計數機)

　

例題: 展開 (3x – 2y)4

SRP-285II 按法:

按 P/V RCL，再按 ← 選擇正確的PROG，再按 = =  

3 = - 2 = 4 = 0 = (0表示第一個係數，答案為81)

↑1 = (1表示第二個係數，答案為 -216)

↑2 = (2表示第 三個係數，答案為 216)

↑3 = (答案為-96)

↑4 = (答案為16)

因此，(3x – 2y)4 = 81x4 – 216x3y + 216x2y2 -96xy3 + 16y4

　

HP 30S 按法:

按 VRCL，再按 ← = = 3 = - 2 = 4 = 0 = (0表示第一個係數，答案為81)

↑1 = (1表示第二個係數，答案為 -216)

↑2 = (2表示第 三個係數，答案為 216)

↑3 = (答案為-96)

↑4 = (答案為16)

因此，(3x – 2y)4 = 81x4 – 216x3y + 216x2y2 -96xy3 + 16y4

　

　

注意事項: 在輸入每一個數據前請先按CL清除之前數據，否則可能出錯，不過若果你輸入的數據的長度等於或長於原先數據的長度，亦可以不按CL。另一個方法是在執行程式之前先按 2nd CL-VAR清除所有數據。
