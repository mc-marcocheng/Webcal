二階矩陣特徵值

程式編寫日期: 2006年9月5日

注意: 輸入程式前請按 MODE 3確定在二次方程模式，紅色的X是方程變數X，亦即是按 2nd )

程式 (17字)

X2 + 0A - BC + AD = AX + DX

STO EQN

註: 儲存程式時會出現 SYN Error，這是正常，程式已儲存。

　

例題: 計算下列矩陣的特徵值。　

         

按 MODE 3 VRCL，再按 ← = = 4 = 1 = 6 = 2 = (顯示5.64575) → (顯示0.35425)

所以特徵多項式為 x2 – 6x + 2 ，特徵值為5.64575及0.35425。

註: 若果使用程式前已在二次方程模式，可以不按 MODE 3。

　

注意事項: 在輸入每一個數據前請先按CL清除之前數據，否則可能出錯，不過若果你輸入的數據的長度等於或長於原先數據的長度，亦可以不按CL。另一個方法是在執行程式之前先按 2nd CL-VAR清除所有數據。
