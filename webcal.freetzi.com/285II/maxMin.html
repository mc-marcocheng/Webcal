二次函數極小/大點及圓形半徑

程式 (共58字，藍色的 - 是負號，按EXP或E可以產生E。):

√( ( A2 + B2 - 4C ) X ( X - 1 ) / 8 ) - ( 4A + E - 70 ) -1 ( B ( X2 - 3X + 2 ) + ( B2 - 4AC ) X ( 2 - X ) )

SAVE PROG1 或 PROG2 (SRP-285II計數機)

STO EQN (HP 30S計數機)

　

例題1: 求函數 y = x2 + 6x + 25 的極小點

SRP-285II 按法:  

按 P/V RCL，再按 ← 選擇正確的PROG，再按 = =

1 = 6 = 25 = 0 = (0表示x值，顯示答案為-3)

↑1 = (1表示y值，顯示答案為16)

　

HP 30S 按法:

按 VRCL，再按 ← = = 1 = 6 = 25 = 0 = (0表示x值，顯示答案為-3)

↑1 = (1表示y值，顯示答案為16)

　

例題2: 求圓的半徑: x2 + y2 – 4x + 6y – 12 = 0。

SRP-285II 按法:  

按 P/V RCL，再按 ← 選擇正確的PROG，再按 = =

- 4 = 6 = - 12 = 2 = (2表示計半徑，顯示圓的半徑為5)

　

HP 30S 按法:

按 VRCL，再按 ← = = - 4 = 6 = - 12 = 2 = (2表示計半徑，顯示圓的半徑為5)

　

注意事項: 在輸入每一個數據前請先按CL清除之前數據，否則可能出錯，不過若果你輸入的數據的長度等於或長於原先數據的長度，亦可以不按CL。另一個方法是在執行程式之前先按 2nd CL-VAR清除所有數據。
