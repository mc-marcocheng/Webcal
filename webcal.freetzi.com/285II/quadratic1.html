一元二次方程

注意: HP 30S需要輸入整個方程式，X是按 2nd ( 。

例題1: 解 x2 - 7x + 12 = 0

SRP-285II 按法:

按 MODE 5 = (選擇QE模式)

1 = - 7 = 12 = (顯示第一個解為4) 再按 → (顯示另一個解為3)

注意輸入數值是負數要按"(-)"這個按鍵，並不是減號。

HP 30S 按法:

若果不是在二次方程模式( QSOLV)，按MODE 3

按出方程 X2 - 7X + 12 = 0

再按 =  (顯示第一個解為4) 再按 → (顯示另一個解為3)

　

例題2: 解 x2 + 6x + 25 = 0

SRP-285II 按法:

按 MODE 5 = (選擇QE模式)

1 = 6 = 25 = (顯示NO REAL SOL，表示無實解)

HP 30S 按法:

若果不是在二次方程模式( QSOLV)，按 MODE 3

按出方程 X2 + 6X + 25 = 0

再按 =  (顯示NO REAL SOL，表示無實解)

　

註1: 答案若為小數，可以用內置分數功能將小數轉換為分數(按F<>D)。

註2: HP 30S除了可以使用X變數外，亦可以是Y變數，輸入的二次方程亦可以不是標準形式。
