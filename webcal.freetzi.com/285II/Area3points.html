三點求三角形面積

程式 (共29字):

0ABC + √( ( AX + YB +X1C - CB - XY - AX1 )2 ) / 2

SAVE PROG1 或 PROG2 (SRP-285II計數機)

STO EQN (HP 30S計數機)

　

例題: 己知三點分別為(2,3)，(4,5)及(3,7)，求三角形面積。

SRP-285II 按法: 

按 2nd CL-VAR  P/V RCL，再按 ← 選擇正確的PROG，再按 = =

2 = 3 = 4 = 5 = 3 = 7 = (顯示面積為3)

　

HP 30S 按法:

按 2nd CL-VAR  VRCL，再按 ← = = 2 = 3 = 4 = 5 = 3 = 7 = (顯示面積為3)
