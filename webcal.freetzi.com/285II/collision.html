彈性碰撞

若已知兩物體質量及碰撞前的速度，程式可以計算經過彈性碰撞後兩物體的速度。

程式編寫日期: 2006年6月28日

程式 (共26字):

( AC / B + 2X - C ) / ( 1 + A / B ) + ( C - X )Y

SAVE PROG1 或 PROG2 (SRP-285II計數機)

STO EQN (HP 30S計數機)

註: 彈性碰撞滿足動量守恆定律及碰撞時沒有動能損耗。

　

例題: 物體A的質量及初速度分別為 5 kg及6 ms-1，物體B最初則靜止，質量為 10 kg，若物體A正面撞向物體B，碰揰為完全彈性碰撞，求A及B碰撞後的速度。
