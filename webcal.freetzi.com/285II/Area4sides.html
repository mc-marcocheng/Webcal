圓內接四邊形四邊求面積(包括希羅公式)

程式 (共40字):

√( ( A - B + C + X ) ( B + C + X - A ) ( A + B - C + X ) ( A + B + C - X ) ) / 4

SAVE PROG1 或 PROG2 (SRP-285II計數機)

STO EQN (HP 30S計數機)

　

例題1: 已知圓內接四邊形的四邊分別為5,12,12及5，求面積。

SRP-285II 按法:

按 P/V RCL，再按 ← 選擇正確的PROG，再按 = =

5 = 12 = 12 = 5 = (顯示面積為60)

 

HP 30S 按法:

按 VRCL，再按 ← = = 5 = 12 = 12 = 5 = (顯示面積為60)

　

例題2: 己知三角形的三邊分別為3，4及5，求面積。

SRP-285II 按法:

按 P/V RCL，再按 ← 選擇正確的PROG，再按 = =

3 = 4 = 5 = 0 = (輸入0代表計算三角形面積，顯示面積為6)

　

HP 30S 按法:

按 VRCL，再按 ← = = 3 = 4 = 5 = 0 = (輸入0代表計算三角形面積，顯示面積為6)

 

注意事項: 在輸入每一個數據前請先按CL清除之前數據，否則可能出錯，不過若果你輸入的數據的長度等於或長於原先數據的長度，亦可以不按CL。另一個方法是在執行程式之前先按 2nd CL-VAR清除所有數據。
