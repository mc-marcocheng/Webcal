聯立三元一次方程通解(應用聯立二元一次方程)

編寫日期: 2006年5月11日

聯立三元一次方程的通解是可以利用聯立二元一次方程求得，請參看以下例子。

　

例題: 試找出下列聯立三元一次方程的通解:

方程: 

x + 2y - 5z = 4

4x - y - 2z = 7

SRP-285II 按法:

按 MODE 4 = 再按 1 = 2 = 5 (輸入z的係數，正負值要相反，暫時不輸入常數項)

4 = - 1 = 2 = (顯示 1，即是x的解，t的係數) → (顯示 2，即是y的解，t的係數)

按 MODE 4 = 再按 = = 4 = (不輸入z的係數，輸入常數項)

= = 7 = (顯示 2，即是x的解，常數項) → (顯示1，即是y的解，常數項)     

HP 30S 按法:

若果不是在聯立二元一次方程模式( LSOLV)，按MODE 2

按出方程 X + 2Y - 5 = 0 , 4X - Y - 2 = 0

再按 = (顯示 1，即是x的解，t的係數) → (顯示 2，即是y的解，t的係數)

再按 == ↑↑修改方程為 X + 2Y = 4 ,4X - Y = 7

再按 = (顯示 2，即是x的解，常數項) → (顯示1，即是y的解，常數項)     
　

因此通解為 : x = t + 2, y = 2t + 1, z = t

 

註: 答案若為小數，可以用內置分數功能將小數轉換為分數(按F<>D)。
