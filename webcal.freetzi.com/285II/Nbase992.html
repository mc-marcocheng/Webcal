基數運算

按鍵說明:

MODE MODE MODE MODE MODE 1: 進入基數模式

DEC: 設定十進制計算

HEX: 設定十六進制計算

SHIFT BIN: 設定二進制計算

SHIFT OCT: 設定八進制計算

BLOCK: 二進制答案會顯示第1組，按一下BLOCK會顯示第2組→第3組→第4組，然後返回第1組。

SHIFT BLOCK: 二進制答案會顯示第1組，按一下SHIFT BLOCK會顯示第4組→第3組→第2組，然後返回第1組。

註: 顯示二進制答案時，最左邊的四個字元用作表示組別，若細小的o字在右邊第1個位置代表第1組，第2個位置代表第2組，第3個位置代表第3組，第4個位置代表第4組。

MODE 1: 返回正常計算模式

　

例題1: 將2610轉為二進制、八進制及十六進制。

先進入基數模式，按 MODE MODE MODE MODE MODE 1 

再按 DEC 26 SHIFT BIN (顯示11010 b，即是110102)

再按 SHIFT OCT (顯示32 o，即是328)

再按 HEX (顯示1A H，即是1A16)

完結後，按 MODE 1 返回正常計算模式。

 

例題2: 計算 1FA16 + 5078，最後答案以十六進制表示。

先進入基數模式，按 MODE MODE MODE MODE MODE 1 

再按 HEX 1FA + OCT 507 = HEX (顯示341 H，即是34116)

完結後，按 MODE 1 返回正常計算模式。

 

例題3: 計算 10112 AND 11002

先進入基數模式，按 MODE MODE MODE MODE MODE 1 

再按 BIN 1011 AND 1100 = (顯示1000b，即是10002)

完結後，按 MODE 1 返回正常計算模式。
