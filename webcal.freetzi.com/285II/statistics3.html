例題1: 若一組數據X的數值 為1, 1, 2, 3, 4, 5，求

(i) 若X=4時的標準分得分，

(ii) 求P(X<4)、P(X<X<4)及P(X>4)。

　

按 MODE 1 再按 → 或 ← 選擇普通統計 1-VAR =

計算前先清除統計記憶，按 MODE 1 再按 ← 選擇 D-CL =

再按 DATA ← = 4 = 再按 DATA = 1↓2↓2↓↓3↓↓4↓↓5↓↓

STATVAR ← (顯示R(t)為0.1855)

再按 ← (顯示Q(t)為0.3145)

再按 ← (顯示P(t)為0.8145)

再按 ← (顯示 t為0.8944)

計算完成後按 Mode 0返回正常計算模式。

所以:

P(X≧4)=0.1855

P(X≦ X≦4)=0.3145

P(X≦4)=0.8145

標準得分=0.8944
