四點求四邊形面積

程式 (共35字):

0ABC + √( ( AX - XY + MC - CB + X1B - AX2 + X2Y - X1M )2 ) / 2

SAVE PROG1 或 PROG2 (SRP-285II計數機)

STO EQN (HP 30S計數機)

　

例題: 己知四點分別為(0,0)，(3,4)，(-1,7)及(-4,3)，求四邊形面積。

SRP-285II 按法: 

按 2nd CL-VAR  P/V RCL，再按 ← 選擇正確的PROG，再按 = =

 0 = 0 = 3 = 4 = - 1 = 7 = - 4 = 3 = (顯示面為25)

　

HP 30S 按法:

按 2nd CL-VAR  VRCL，再按 ← = = 0 = 0 = 3 = 4 = - 1 = 7 = - 4 = 3 = (顯示面 積為25)

 

註: 四點要依須時針或逆時針方向輸入，否則不能正常計算四邊形面積。
