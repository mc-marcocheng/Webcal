已知斜率求二次曲線的切線

程式編寫日期: 2007年12月29日

若已知切線的斜率，程式可以計算切線的Y-截距(二次曲線方程式為 Ax2 + Bxy + Cy2 + Dx + Ey + F = 0)，另外輸入數據為整數(或分數)及答案為有理數時，答案會以分數形式表示，建議將計數機預先設定為假分數形式表示(按 SHIFT SETUP → → 2 )。

程式 (101 bytes)

?→A: ?→B: ?→C: ?→M : ?→X: ?→Y: ?→D:

A + BD + CD2→A: B + 2CD→B: XDM+:

B2 - 4AC→C: 2AX - BM→D: D2 - CM2 + 4ACY:

(√( Ans ) + D)┘C→A◢ 2D┘C - Ans→B

　

例題: 一直線的斜率為–2，它與圓 x2 + y2 + 2x + 4y = 0相切，求直線方程。

按 Prog 1 再按 1 EXE 0 EXE 1 EXE 2 EXE 4 EXE 0 EXE - 2 EXE (顯示切線的y-截距為-9)

EXE (顯示另一切線的y-截距為1)

所以兩切線分別為 y = - 2x - 9 及 y = - 2x + 1

程式執行完成後，按 RCL A及RCL B分別顯示兩直線的y-截距。
