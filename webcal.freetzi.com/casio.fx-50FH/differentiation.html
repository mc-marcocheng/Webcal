微分計算

程式可計算一個函數的函數值、第一及第二導數的近似值。 程式二較長，但會根據變數X的大小，決定合適的△X，因此能克服現時很多同類程式的問題，即是改善了當X的值很大或很細時出現誤差較大的問題。

程式更新日期: 2010年6月5日

注意: E是按 EXP。

程式一 (71 bytes，不包括綠色的函數方程)

CltrMemory: ?→X: E4→Y:

While Ans: X3 - 2X - 1→C: X + Y-1(1 - 3M→X:

D→A: B→D: C→B: M≠2M+ : WhileEnd:

A◢ . 5Y(D - C→B◢ Y2(D + C - 2A→C

　

程式二(79 bytes，不包括綠色的函數方程)

CltrMemory: ?→X: E4 ÷ (X + (X=0→Y:

While Ans: X3 - 2X - 1→C: X + Y-1(1 - 3M→X:

D→A: B→D: C→B: M≠2M+ : WhileEnd:

A◢ . 5Y(D - C→B◢ Y2(D + C - 2A→C

　

註: 綠色的X3 - 2X - 1是函數方程(變數是X)，若果想計算其它方程，只要修改綠色的部份。

　

例題: 計算 f(x) = x3 - 2x - 1 在 x = 1的第一及第二導數的近似值。

按 Prog 1  再按 1 EXE (顯示 f(1)為 -2) EXE (顯示f'(1)為1.00000001) EXE (顯示f''(1)為6)

　

計算完結後，按 RCL A、RCL B及RCL C分別顯示函數值、第一導數及第二導數的近似值。

　

返回 CASIO fx-50FH、fx-3650P II、fx-50FH II及fx-50F PLUS 程式集
