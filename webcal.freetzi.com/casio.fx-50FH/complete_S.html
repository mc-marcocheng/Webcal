頂點功能求配方法算式

程式編寫日期: 2008年1月20日

程式可以計算出二次函數的頂點座標，由頂點座標其實可以直接求出二次函數配方法的算式。

若頂點座標為 (h, k) ，則二次函數配方法算式為:

　

ax2 + bx + c = a(x - h)2 + k

　

例題: 二次函數為f(x) = 2x2 + x - 3，求配方法的算式。

假設使用一元二次方程(I)的程式

按 Prog 1  再按 2 EXE 2 EXE -3 EXE (顯示第一個實數根為1) EXE (顯示第二個實數根為-3/2)

再按 RCL C (顯示-1/4) RCL M (顯示-25/8，即頂點為 (-1/4, -25/8) )

所以

2x2 + x - 3 = 2(x - (-1/4) )2 + (- 25/8)

2x2 + x - 3 = 2(x + 1/4 )2 - 25/8
