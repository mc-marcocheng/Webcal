元素的族及周期

程式可以根據元素的原子序(atomic number)計算出元素的族及周期(Group and Period)。

程式是參考網友 Mike 提供的程式作為基礎，進行修改及強化功能，使程式亦可以計算過渡金屬(Transition metals)、鑭系元素(Lanthanoids)及錒系元素(Actinoids)的情況。

程式編寫日期: 2008年6月14日

注意: 藍色程式碼 o 是按 o,,,

程式 (154 bytes)

ClrMemory: ?→A:  1→B: (A=1) + 18(A=2M+ => Goto 0:  

2→X: While Ans: 1 + B→B: Fix 0: 2Rnd(B÷2 + . 5)2→Y:

Norm 1:  X + Y→D: A>D => D→X: WhileEnd: A - X→D:

D - Y + 18M+: Y=32 => D>3 => 18>D => MM-: Y=32 =>

D=3 => D→M:2≧D => D→M: 18≧A => D + 10(D>2→M:

Lbl 0: Bo + M÷60o 

　

例題: 求原子序52的碲元素 Te 族及周期數。

按 Prog 1 再按 52 EXE (顯示5o16o0，即為周期為5及元素為16族  )

　

註: 族(Group)數值由1至18，若果數值出現0，表示族數不適用(n/a)。
