一般操作問題

問題:

1. 如何計算次方(例如: 210)及開多次方(例如: 5√32)的問題？

2. 小數位及有效數字是如何設定及取消？

3. 計數機的操作模式如何設定?

4. 三角函數的度數及弧度是如何設定？

5. 如何將計算的結果儲存在計數機中？

6. 為何計算平均數及標準差總是很容易出錯？

7. 如何輸入有10的指數部分的數值(例如: 2.1×1011)？

8. 修改算式時，如何可以將新的字元插進算式中?

9. 計算出現錯誤 (ERROR) 不知如何做？

　

解答:

1. 如何計算次方(例如: 210)及開多次方(例如: 5√32)的問題？

答: 使用計數機上 ^ 及 x√   的功能，

例如:計算210 

按 2  ^  10 EXE  (顯示算式 2^10 及答案1024)

計算5√32

按 5  SHIFT  x√  32 (顯示算式 5x√32= 及答案 2)

　

2. 小數位及有效數字是如何設定及取消？

答: 小數位設定按 MODE MODE MODE MODE MODE 1  再按數字(0至9)設定顯示小數位

有效數字設定按 MODE MODE MODE MODE MODE 2 再按數字(0至9)設定顯示位數

取消小數位或有效數字設定按 MODE MODE MODE MODE MODE 3 再按 1 或 2

註: Norm 1表示對大於10位的整數及小數位數多於2位的小數，指數記法將被自動採用。

      Norm 2表示對大於10位的整數及小數位數多於9位的小數，指數記法將被自動採用。

     

3. 計數的操作模式如何設定?

答:

正常計模式:     按 MODE 1 (無特別的細字顯示)

複數模式:         按 MODE 2 (有 CMPLX 細字顯示)

基數模式:         按 MODE MODE 3 (計數機右邊有 H或o或d或b 細字顯示所用進制)

普通統計模式: 按 MODE MODE 1 (有 SD 細字顯示)

回歸模式 (有 REG 細字顯示)

線性回歸模式:  按 MODE MODE 2 1

對數回歸模式:  按 MODE MODE 2 2

指數回歸模式:  按 MODE MODE 2 3

冪次回歸模式:  按 MODE MODE 2 → 1

倒數回歸模式:  按 MODE MODE 2 → 2

二次回歸模式:  按 MODE MODE 2 → 3

　

4. 三函數的度數及弧度是如何設定？

答:

度數設定:     按 MODE MODE MODE MODE 1

弧度設定:     按 MODE MODE MODE MODE 2

百分度設定: 按 MODE MODE MODE MODE 3

　

5. 如何將計算的結果儲存在計數機中？

答: 可以儲存數值的字母記憶有A、B、C、D、X、Y及M，例如要將數值儲存到記憶A，只要按 SHIFT STO A，要顯示記憶A的數值按 RCL A，要使用記憶A計算按ALPHA A 。

　

6. 為何計算平均數及標準差總是很容易出錯？

答: 出現錯誤的原因可能很多，可以注意下列幾點，看看情況可否改善。

(1) 在輸入新數據前，必定要先清除原先的統計記憶(Stat clear)按 SHIFT MODE 1，否則計算很可能出錯，不過在按 MODE MODE 1除了切換到SD模式外，同時亦會清除原先的統計記憶。

(2) 由於可能要輸入不少的統計數據，輸入時要較為小心，輸入完成後計數機會同時顯示輸入數據的總數，請核對真實數據的數目是否相同，若果不相同表示輸入錯誤。

　

7. 如何輸入有10的指數部分的數值(例如: 2.1×1011)？

答: 使用 EXP 這個按鍵，例如: 2.1×1011 按 2.1 EXP 11

　

8. 修改算式時，如何可以將新的字元插進算式中?

答: 按 SHIFT INS (亦即是按 SHIFT DEL) 即可將游標改為插入游標，再按 SHIFT INS 會再返回至普通游標 (新輸入的字元會覆蓋舊的字元)。

　

9. 計算時出現錯誤 (ERROR) 不知如何做？

答: 可以使用錯誤指示器 (Error locator)。當出現錯誤時(ERROR)，按 → 或 ←鍵會調出計算式，而游標即會停留在錯誤出現的位置上，只要找出錯誤原因及改正，再執行算式即可。另外出現錯誤時，會顯示錯誤的類型，例如: Syntax ERROR (表示算式中含有錯誤)，Math ERROR (表示計算結果超出可計算範圍 或 輸入數值超出輸入範圍 或 計算一個不合理運算 (例如: 除以零)，這些資都可以幫助找出錯誤的原因。
