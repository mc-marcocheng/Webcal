回歸計算

按鍵說明:

(A) 回歸模的選擇

MODE MODE 2 1 (進入線性回歸模式: yEXE A + Bx)

MODE MODE 2 2 (進入對數回歸模式: yEXE A + B ln x)

MODE MODE 2 3 (進入指數回歸模式: yEXE A eBx)

MODE MODE 2 → 1 (進入乘方回歸模式: yEXE A xB)

MODE MODE 2 → 2 (進入逆回歸模式: yEXE A + B/x)

MODE MODE 2 → 3 (進入二次回歸模式: yEXE A + Bx +Cx2)

(B) 回歸變數

SHIFT MODE 1 EXE (清除統計記憶)

SHIFT S-SUM 1 EXE (顯示數據的平方總和 Σx2)

SHIFT S-SUM 2 EXE (顯示數據的總和 Σx)

SHIFT S-SUM 3 EXE (顯示數據的數量 n)

SHIFT S-SUM → 1 EXE (顯示數據的數量 Σy2)

SHIFT S-SUM → 2 EXE (顯示數據的數量 Σy)

SHIFT S-SUM → 3 EXE (顯示數據的數量 Σxy)

SHIFT S-SUM → → 1 EXE (只限二次回歸模式，顯示數據的數量 Σx3)

SHIFT S-SUM → → 2 EXE (只限二次回歸模式，顯示數據的數量 Σx2y)

SHIFT S-SUM → → 3 EXE (只限二次回歸模式，顯示數據的數量 Σx4)

SHIFT S-VAR 1 EXE (顯示x數據的平均數 x)

SHIFT S-VAR 2 EXE (顯示x數據的標準差 xσn)

SHIFT S_VAR 3 EXE (顯示x數據的樣本標準差xσn-1)

SHIFT S-VAR → 1 EXE (顯示y數據的平均數 y)

SHIFT S-VAR → 2 EXE (顯示y數據的標準差 yσn)

SHIFT S_VAR → 3 EXE (顯示y數據的樣本標準差yσn-1)

SHIFT S_VAR → → 1 EXE (顯示回歸係數A)

SHIFT S_VAR → → 2 EXE (顯示回歸係數B)

SHIFT S_VAR → → 3 EXE (顯示回歸係數 r 或 C)

輸入數據的方法: [x數據] , [y數據] DT 或  [x數據] , [y數據] ; [數量] DT

注意: 當改變計數機模式(包括改變到相同的統計模式)，統計記憶會自動被清除。

　

例題: 利用線性回歸方法，找出下列數據的線性關係，估計當xEXE10，y的數值?估計當yEXE38，x的數值?
x 2 2 13 19 19 19 15
y 5 5 26 40 40 40 30

　

　

按 MODE MODE 2 1 (進入線回歸模式及清洗統計記憶)

再按 2 , 5 DT DT 13 , 26 DT 19 , 40 SHIFT ; 3 DT 15 , 30 DT

SHIFT S_VAR → → 1 EXE (顯示A為0.556992724)

SHIFT S_VAR → → 2 EXE (顯示B為 2.046079224)

SHIFT S_VAR → → 3 EXE

(顯示 r為0.998615112，這個數值(correlation coefficient)越近1或-1，代表越接近線性關係)

10 SHIFT S_VAR → → → 2 EXE (顯示21.01778496)

38 SHIFT S_VAR → → → 1 EXE (顯示18.29988147)

　

因此直線關係為 y EXE 0.556992724 +  2.046079224 x

當xEXE10，y EXE 21.01778496

當yEXE38，x EXE 18.29988147

完成計算後，按 MODE 1 返回正常模式。

注意: 若計算新一組統計問題，請按SHIFT MODE 1 EXE (清除統計記憶)
