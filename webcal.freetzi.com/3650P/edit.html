輸入/修改程式的步驟

　

1. 若果是輸入新程式，請查看現時的操作模式是否與程式所用的相同，若果不同請先進入該模式。

COMP mode (基本計算模式，計數機無特別顯示): 按 Mode 1

CMPLX mode (複數模式，計數機會顯示細小的CMPLX): 按 Mode 2

SD mode(標準統計模式，計數機會顯示細小的SD): 按 Mode Mode 1

REG mode (回歸統計模式): 按 Mode Mode 2 ，此時有三個選擇(1. Lin 2. Log 3. Exp)，

再按數字進入其中一個模式，若果不是這三個，則按 Replay 右鍵，此時有另外三個選擇 (1. Pwr 2. Inv 3. Quad)，按數字選擇其中一個模式，此時計數機會顯示REG

　

2. 按 Mode Mode Mode 1 ，此時顯示 P-1234 及程式餘下可用的空間。再選擇編寫程式的位置(按1,2,3或4)，進入程式的Edit 模式。

　

3. 輸入新程式或修改舊程式。修改的方法如下:

按 Shift DEL (INS)計數機的浮標會變成方形， 表示會將新指令加插在程式中，舊有的指令不會被刪除，而會向右移。此時按一下DEL，只會刪除左方一個字元。

再按 Shift DEL (INS)計數機的浮標合變回底横線，表示新加入的指令會覆蓋舊有指令，此時按一下DEL，只會刪除浮標位置的字元。

　

4. 輸入或修改完成後，請查看程式所用實際空間是否與程式大小相同。若果不同表示很有可能輸入有錯誤，請小心檢查。

　

5. 按 Mode Mode Mode 2 或 直按 ON 跳出Edit Mode，返回RUN模式。

　

6. 按 Prog 再選擇執行程式(1,2,3或4)，以例題作測試，若結果與例題不同，表示輸入有錯誤，請再進入程式的Edit模式小心檢查。若果測試程式時出現Error，請按Replay 右/左鍵，計數機會直按進入Edit模式，並且浮標的位置就是出現問題的位置。另外亦應留意錯誤訊息，對於了解程式的問題及除錯都會有幫助。

　

刪除程式

按 MODE MODE MODE 3 ，此時會顯示 Clear Prog 及程式餘下可用的空間。再選擇刪除程式的位置(按1,2,3或4)。
