改善歐拉法微分方程計算

程式編寫日期: 2007年5月4日 修改日期: 2008年8月7日

程式 (81 bytes，不包括綠色的函數方程 )

Mem clear: ?→X: ?→Y: ?→A: ?→M: M-1(A - X→A:

Lbl 0: 1 + 2X + Y→B: CM- => Y + . 5A(B - D→Y:

C=0→C => X + A→X: C => Y + AB→Y: C => B→D:

M => Goto 0: Y

注意: 若果想計算其它微分方程，只要修改綠色的部份即可(變數x使用記憶X，變數y使用記憶Y)。

　

例題: 利用改善歐拉法計算微分方程 dy/dx = 1 + 2x + y, y(1)=2的解在 x =3 的近似值，其中區間數目為12。

按 Prog 1 再按 1 EXE 2 EXE 3 EXE 12 EXE (顯示 y(3)的近似值為42.30210573)

計算完成後，按 RCL Y 顯示近似值答案。
