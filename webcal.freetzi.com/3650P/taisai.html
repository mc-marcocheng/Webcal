E2→M◢ Lbl 0: M→D: log M: ?→X: Fix 0:

Ans→C: Norm 1: A=B => A+10=Y => 9XM+:

B=C => B+10=Y => 9XM+: A=C => A+10=Y => 9XM+:

玩法:

1. 顯示本金 $100

2. 輸入投注金額 (投注金額小於本金)

3. 輸入欲投注的項目 (整數1至6分別為圍一至圍六、7為全圍、8,9分別為小及大、11至16分別為雙骰1至6)

4.顯示第一顆骰子滾出的數值

5.顯示第二顆骰子滾出的數值

6.顯示第三顆骰子滾出的數值

7.顯示該次的盈利或虧損

8.顯示最後的本金

程式會重複步驟2至8，直至本金為0時顯示Math ERROR

賠率的計算方法如下:

中大小:+1倍

中雙骰:+8倍

中圍骰:+150倍

中全圍:+24倍
