元素的族、周期及外層電子數目(I)

程式可以根據元素的原子序(atomic number)計算出元素的族(Group)、周期(Period)及外層電子數目(Number of electron on outermost shell)。

程式是參考網友 Mike 提供的程式，進行修改及增加計算外層電子數目功能(包括過渡金屬(Transition metals)、鑭系元素(Lanthanoids)及錒系元素(Actinoids)的情況)。

程式編寫日期: 2008年6月16日

注意: 藍色程式碼 o 是按 o,,,

程式 (262 bytes)

?→A: 2→X: 1→B: (A=1) + 18(A=2→C => Goto 0:

Lbl 1: 1 + B→B: B÷2 - . 5: Fix 0: Rnd: Norm 1:

2(Ans + 1)2→Y: X + Y→D: A>D => D→X => Goto 1:

A - X→D:  D - Y + 18→C: Y=32 => D>3 => 18>D => 0→C:

Y=32 => D=3 => D→C: 2≧D => D→C:

18≧A => D + 10(D>2→C: Lbl 0: C - 10(C>12→M:

C>2 => 13>C => 2→M: C=0 => 2→M: 6(A=2M-:

A=24M-: A=29M-: A=46M-: A=78M-: A=79M-: A=110M-:

A=111M-: A>40 => 48>AM-: Bo + C÷60o + M÷602 o  

　

例題: 求原子序52的碲元素 Te 族及周期數。

按 Prog 1 再按 52 EXE (顯示5o16o6，即為周期為5、元素為16族及外層電子數目為6)

　

註: 族(Group)數值由1至18，若果數值出現0，表示族數不適用(n/a)。
