例題: 若 f(x) = x3 + 3x - 2，求 f(0), f(1), f(2)及f(3)

方法一:

按 0 EXE 再按 Ans  x3 + 3Ans - 2 EXE (顯示 f(0)為 - 2)

1 EXE ↑EXE (顯示 f(1)為 2)

2 EXE ↑EXE (顯示 f(2)為 12)

3 EXE ↑EXE (顯示 f(3)為 34)

　

方法二:

按 0 SHIFT STO X

再按 ALPHA X  x3 + 3 ALPHA X - 2 EXE (顯示 f(0)為 - 2)

1 SHIFT STO X ↑EXE (顯示 f(1)為 2)

2 SHIFT STO X ↑ EXE(顯示 f(1)為 2)

3 SHIFT STO X ↑EXE (顯示 f(3)為 34)
