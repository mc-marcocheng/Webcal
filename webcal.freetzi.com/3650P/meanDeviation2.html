平均偏差

程式編寫日期: 2006年8月30日

程式用作計算平均偏差。可以單獨使用(需要輸入平均值)或配合內置普通統計(I)功能。

程式需要在 SD 模式下執行，因此在輸入程式前請先按 Mode Mode 1 進入SD模式。

注意: 藍色的英文字為統計模式中的變數(n 按 shift 1 3 ，x為平均x 按 shift 2 1)

程式 (48 bytes，使用記憶為A、B及M)

n => x→M: n=0 => ?→M: Stat clear: Lbl 0: ?→A:

1→B: ?→B: A - M: √Ans2 ; B DT: x→A: Goto 0

　

例題1: 計算以下一組數值75, 80, 80, 85, 85, 85, 85, 88的平均數、標凖差及平均偏差。

按 MODE MODE 1 75 DT 80 DT DT 85 ; 4 DT 88 DT

SHIFT 2 1 EXE (顯示平均數為82.875) SHIFT 2 2 EXE(顯示標凖差為3.919)

再按 Prog 1  75 EXE EXE 80 EXE 2 EXE 85 EXE 4 EXE

88 EXE EXE (顯示平均偏差為3.40625)

計算完結後按 MODE 1 返回正常計算模式

　

例題2: 已知組數的平均值為82.875，組 的數據分別為75, 80, 80, 85, 85, 85, 85 及88，求平均偏差。

假設在 Comp 模式

按 Prog 1 再按 82.875 EXE (先輸平均值)

75 EXE EXE 80 EXE 2 EXE 85 EXE 4 EXE

88 EXE EXE (顯示平均偏差為3.40625)

計算完結後按 MODE 1 返回正常計算模式

　

註: 若執行程式時並非在SD模式或沒有任統計數據，程式會先要求輸入一個平均值(M?)，然後再輸入數據。若果已有統計數據，程式會自動選取統計數據的平均值作為計算的平均值。
