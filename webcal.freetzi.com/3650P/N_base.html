基數運算

編寫日期: 2006年2月1日

　

例題1: 將2610轉為二進制、八進制及十六進制。

先進入基數計算模式，按 MODE MODE 3

再按 DEC 26 EXE BIN (顯示11010 b，即是110102)

再按 OCT (顯示32 o，即是328)

再按 HEX (顯示1A H，即是1A16)

基數計算完結後，按  MODE 1 返回正常計算模式。

　

例題2: 計算 1FA16 + 5078，最後答案以十六進制表示。

先進入基數計算模式，按 MODE MODE 3

再按 HEX 1 F A EXE OCT + 507 EXE HEX (顯示341 H，即是34116)

基數計算完結後，按 MODE 1 返回正常計算模式。

另一種按法:

先進入基數計算模式，按 MODE MODE 3

再按 HEX 1 F A + LOGIC LOGIC LOGIC 4  507 EXE (顯示341 H，即是34116)

基數計算完結後，按 MODE 1 返回正常計算模式。

　

例題3: 計算 10112 AND 11002

先進入基數計算模式，按 MODE MODE 3

按 BIN 1011 LOGIC 1  1100 = (顯示1000b，即是10002)

基數計算完結後，按 MODE 1 返回正常計算模式。
