衝擊反應

程式編寫日期: 2007年4月14日

程式 (最少39 bytes)

Mem clear: 1→X: Lbl 0: X - 0.5Y + 2A→D◢

B→C: A→B: D→A: Y→M: X→Y: 0→X: Goto 0

　

註: 綠色的 X - 0.5Y + 2A是函數方程，若果想計算 其它問題，只要修改綠色的部份，修改時注變數A代表 y[n-1]，變數B代表 y[n-2]，變數C代表 y[n-3]，y[n-4] 或之前的項假設不存在，變數X代表 x[n]，變數Y代表 x[n-1]，變數M代表 x[n-2]，x[n-3]或之前的項假設不存在。

　

Example: A filter is described by the following difference equation:

y[n] = x[n] - 0.5 x[n-1] + 2 y[n-1]

Evaluate the impulse response of the filter (i.e. evaluate y[0], y[1], y[2], y[3], y[4], y[5]).

Press Prog 1 (read-out: 1) EXE (read-out: 1.5) EXE (read-out: 3)

EXE (read-out: 6) EXE (read-out: 12) EXE (read-out: 24)

　

Therefore, the result is:

y[0] = 1

y[1] = 1.5

y[2] = 3

y[3] = 6

y[4] = 12

y[5] = 24
