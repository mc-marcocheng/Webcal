試位法

程式新版 (舊版依然保留在網頁下方)

新版程式編寫日期: 2006年1月3日

程式 (最少 53 bytes)

Mem clear: Lbl 0: ?→X: X3 - 2X - 1→C◢

C>0 => C→Y => X→A: 0>C => C→D => X→B:

(AD - BY) ÷ (D - Y→X: Goto 0

　

註: 綠色的X3 - 2X - 1是函數方程(變數是X)，若果想計算其它 方程，只要修改綠色的部份。
