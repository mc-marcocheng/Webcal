回歸計算

按鍵說明:

(A) 回歸模的選擇

MODE 3 1 (進入線性回歸模式: y= A + Bx)

MODE 3 2 (進入對數回歸模式: y= A + B ln x)

MODE 3 3 (進入指數回歸模式: y= A eBx)

MODE 3 → 1 (進入乘方回歸模式: y= A xB)

MODE 3 → 2 (進入逆回歸模式: y= A + B/x)

MODE 3 → 3 (進入二次回歸模式: y= A + Bx +Cx2)

(B) 回歸變數

Alpha CLR 1 = (清除統計記憶)

Shift S-SUM 1 = (顯示數據的平方總和 Σx2)

Shift S-SUM 2 = (顯示數據的總和 Σx)

Shift S-SUM 3 = (顯示數據的數量 n)

Shift S-SUM → 1 = (顯示數據的數量 Σy2)

Shift S-SUM → 2 = (顯示數據的數量 Σy)

Shift S-SUM → 3 = (顯示數據的數量 Σxy)

Shift S-SUM → → 1 = (只限二次回歸模式，顯示數據的數量 Σx3)

Shift S-SUM → → 2 = (只限二次回歸模式，顯示數據的數量 Σx2y)

Shift S-SUM → → 3 = (只限二次回歸模式，顯示數據的數量 Σx4)

Shift S-VAR 1 = (顯示x數據的平均數 x)

Shift S-VAR 2 = (顯示x數據的標準差 xσn)

Shift S_VAR 3 = (顯示x數據的樣本標準差xσn-1)

Shift S-VAR → 1 = (顯示y數據的平均數 y)

Shift S-VAR → 2 = (顯示y數據的標準差 yσn)

Shift S_VAR → 3 = (顯示y數據的樣本標準差yσn-1)

Shift S_VAR → → 1 = (顯示回歸係數A)

Shift S_VAR → → 2 = (顯示回歸係數B)

Shift S_VAR → → 3 = (顯示回歸係數 r 或 C)

Shift S-PTS 1 = (顯示數據X的最小值)

Shift S-PTS 2 = (顯示數據X的最大值)

Shift S-PTS → 1 = (顯示數據X的最小值)

Shift S-PTS → 2 = (顯示數據X的最大值)

　

輸入數據的方法: [x數據] , [y數據] Data 或  [x數據] , [y數據] ; [數量] Data

注意: 當改變計數機模式(包括改變到相同的統計模式)，統計記憶會自動被清除。

　

例題: 利用線性回歸方法，找出下列數據的線性關係，估計當x=10，y的數值?估計當y=38，x的數值?
x 2 2 13 19 19 19 15
y 5 5 26 40 40 40 30

　

　

按 MODE 3 1 (進入線回歸模式及清洗統計記憶)

再按 2 , 5 Data Data 13 , 26 Data 19 , 40 Shift ; 3 Data 15 , 30 Data

Shift S_VAR → → 1 = (顯示A為0.556992724)

Shift S_VAR → → 2 = (顯示B為 2.046079224)

Shift S_VAR → → 3 =

(顯示 r為0.998615112，這個數值(correlation coefficient)越近1或-1，代表越接近線性關係)

10 Shift S_VAR → → → 2 = (顯示21.01778496)

38 Shift S_VAR → → → 1 = (顯示18.29988147)

　

因此直線關係為 y = 0.556992724 +  2.046079224 x

當x=10，y = 21.01778496

當y=38，x = 18.29988147

完成計算後，按 MODE 1 返回正常模式。
