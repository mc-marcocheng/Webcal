一元二次方程(應用內置二次線性回歸功能)

　

例題: 解 x2 - 7x + 12 = 0

設f(x)= x2 - 7x + 12

當 x = 0, f(0) = 12, 當 x = 1, f(1) = 6, 當 x = -1, f(-1)=20.

　

因此按鍵的方法如下:

MODE 3 → 3   0 ， 12 Data 1 ， 6 Data - 1 ，20 Data

0 Shift 9 ← 1 = (顯示第一個解是4)

0 Shift 9 ← 2 = (顯示第二個解是3)

　

註1:如果答案顯示Math error，表示無實根。

註2:答案若為小數，可以用內置分數功能嘗試將小數轉換為分數(按a b/c 或 2ndF d/c)，

即使是簡單的分數(例如:1/3)，亦不一定會成功。

註3:如果想重新解另一方程， 可以重覆之前的步驟即可，或者將步驟中按 MODE 3 → 3 改為按 Alpha CLR 1 =。
