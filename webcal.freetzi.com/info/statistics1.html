普通統計(I)

按鍵說明:

MODE 2 (進入統計模式)

Alpha CLR 1 = (清除統計記憶)

Shift S-SUM 1 = (顯示數據的平方總和 Σx2)

Shift S-SUM 2 = (顯示數據的總和 Σx)

Shift S-SUM 3 = (顯示數據的數量 n)

Shift S-VAR 1 = (顯示數據的平均數 x)

Shift S-VAR 2 = (顯示數據的標準差 σn)

Shift S_VAR 3 = (顯示數據的樣本標準差σn-1)

Shift S-PTS 1 = (顯示數據的最小值)

Shift S-PTS 2 = (顯示數據的最大值)

輸入數據的方法: [數據] Data 或 [數據] ; [數量] Data

注意: 當改變計數機模式(包括改變到相同的統計模式)，統計記憶會自動被清除。

　

例題: 計算以下一組數值75, 80, 80, 85, 85, 85, 85的平均數及標凖差。

按 MODE MODE 1 (進入統計模式及清除統計記憶)

再按 75 Data 80 Data Data 85 ; 4 Data

Shift S-VAR 1 = (顯示平均數為82.14285714)

Shift S-VAR 2 = (顯示標凖差為3.642156795)

Shift S-PTS 1 = (顯示最小值為75)

Shift S-PTS 2 = (顯示最大值為85)

　

完成計算後，按 MODE 1 返回正常模式。

注意: 若計算新一組統計問題，請按Alpha CLR 1 = (清除統計記憶)
