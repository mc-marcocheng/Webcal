角度單位轉換

編寫日期: 2006年8月20日

預設角度單位設定按:

度數:     按 MODE MODE 1

弧度:     按 MODE MODE 2

百分度: 按 MODE MODE 3

　

角度單位轉換功能可以將顯示數值轉換為相應的預設角度單位。

度數轉換為預設單位:     按 Shift DEG→ 1 =

弧度轉換為預設單位:     按 Shift Ans 2 =

百分度轉換為預設單位: 按 Shift Ans 2 =

　

例題: 將4.25弧度轉換為度。

假設預設單位為度，若果不是度單位，請按 MODE MODE 1

按 4.25 Shift Ans 2 = (第一行顯示4.25r，第二行顯示答案 243.5070629)
