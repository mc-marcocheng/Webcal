迭代法

以下例子說明如何建立迭代式或牛頓迭代式，從而計算出方程一個根的近似值。

牛頓迭代式:  xn+1 = xn - f(xn) / f'(xn)

　

例題1: 利用迭式 x = 2-1(x3 - 1)，計算方程 x3 – 2x – 1 = 0 的其中一個根，以0為開始的數值 。 

按 0 = (將 0 儲存在答案記憶)

再按出算式 2-1 ( Ans3 - 1 )

再按 = (顯示第1近似值為 - 0.5)

再按 = (顯示第2近似值為 - 0.5625)

再按 = (顯示第3近似值為 - 0.588989257)

再按 = (顯示第4近似值為 - 0.602162644)
