函數計算

編寫日期: 2006年2月23日

透過編寫簡單程式可以重複計算函數值，編寫方法很簡單，先按出函數因變數的數值，再按 MODE EXP P1 (或 SHIFT P2)，然後編寫程式，要注意程式的第一步必為 Min，接著好似一般計算，計算函數值即可，最大分別是需要因變數的數值時，按MR提取數值即可，計算出答案時即代表編寫完結，此時再按 MODE . 終結即可，以下例子可以作為參考。

　

例題: 若 f(x) = x3 + 3x - 2，求 f(0), f(1), f(2)及f(3)

注意:輸入程式前先按出因變數的其中一個 0

按 0 MODE EXP P1
Min xy 3 + 3
× MR - 2 =
MODE . 　 　 　 　

(輸入程式完結時，亦同時會顯示f(0)為 - 2)

再按 1 P1 (顯示f(1)為2) 2 P1 (顯示f(2)為12) 3 P1 (顯示f(3)為34)
