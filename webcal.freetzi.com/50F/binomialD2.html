二項分佈(II)

程式編寫日期: 2006年2月4日

程式長度: 29步 (注意: A是按 FMLA a b/c，B是按 FMLA  。'''，C是按 FMLA hyp)
SHIFT 2 Kin + 3 Kout 1 Kin 5 nCr
Kout 2 Kin - 5 × 1 Kin 4
Kout 3 Kin - 4 xy Kout 2 ×
Kout 4 xy Kout 5 = Kin + 6
Kin 2 B Kout 1 x > 0 1
Kin - 6 A C x > 0 MODE .

注意: 執行程式前請按SHIFT KAC。程式除了計算答案外，亦會將答案累加在K6中。

　

例題1: 若X ~ Bin(9,0.5)，求P(X=4)。

按 SHIFT KAC (必要) P1  4 RUN 9 RUN . 5 RUN (顯示答案為0.24609)

　

例題2: 若X ~ Bin(9,0.5)，求P(X=4)，P(X=6)，P(X=7)及P(X=4)+P(X=6)+P(X=7)。

按 SHIFT KAC (必要) P1  4 RUN 9 RUN . 5 RUN (顯示P(X=4)為0.24609)

6 RUN (顯示P(X=6)為0.16406)

7 RUN (顯示P(X=7)為0.07031)

Kout 6 (顯示P(X=4)+P(X=6)+P(X=7)為0.48047)
