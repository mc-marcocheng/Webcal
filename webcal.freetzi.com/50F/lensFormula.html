透鏡公式/球面鏡公式

　

程式編寫日期: 2006年5月12日

程式可計算以下兩種情況的問題:

(1)    已知透鏡或球面鐿的焦距、物距及物高，求像距及像高

(2)    已知透鏡或球面鏡的焦距、像距及像高，求物距及物高

注意: 程式中的公式採用實為正 (real is positive) 的規定: 即如果物、像或焦點是實的，物距、像距及焦距等為正值；如它們是虛的，取負值。

　

程式長度: 13 步

　
A 2 Kin 4 B 3 Kin - 4 Kout 4
+ / - Kin ÷ 1 C 4 Kout 1 Kin × 2
Kin × 3 B Kout 3 MODE . 　

　

例題: 一凸透鏡的焦距為20 cm，物體放在離它25 cm處，物高為1 cm，求像距及像高。

 

按 P1  20 RUN 25 RUN 1 RUN (顯示像距為 100 cm) 再按 RUN (顯示像高為 4 cm)
