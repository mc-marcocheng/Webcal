小數轉換分數(I)

程式新版 (舊版保留在網頁下方)

程式修改日期: 2006年2月17日

這個程式較舊版的程式更簡短，速度亦快了一點。

注意: 在輸入程式前請先按 1 SHIFT Kin 4 MODE 1 DEC MODE 0 確保程式能正確輸入。

程式長度: 14步 (不配合一元二次方程、二次函數的極大/極小值或複數根程式)
1 Kin + 1 Kout 1 × Kout 4
- Kin 2 MODE 1 MODE 0 =
x2 x > 0 B Kout 1 MODE .

　

例題1: 將0.75化為分數。

按 SHIFT KAC (必要) 0.75 SHIFT Kin 4 P1 (顯示分子為3) RUN (顯示分母為4) 

　

註: 這個程式亦可略作修收，以配合內置一元二次方程 、二次函數的極大/極小值或複數根程式，求分數根(有理數)。

程式長度: 17步 (注意: B是按 FMLA  。''')

注意: 在輸入程式前請先按 1 SHIFT Kin 4 MODE 1 DEC MODE 0 確保程式能正確輸入。
1 Kin + 1 Kout 1 × Kout 4
- Kin 2 MODE 1 MODE 0 =
x2 x > 0 B Kout 5 Kin 4
0 x←→k 1 MODE . 　 　

　

例題2: 解 21x2–10x + 1 = 0

按 1 FMLA 21 RUN 10 +/- RUN 1 RUN (顯示0.33333) RUN (顯示0.14286)

0 SHIFT Kin 1 (必要) P1 (顯示第一個根分子為1) RUN (顯示第一個根分母為3)

P1 (顯示第二個根分子為1) RUN (顯示第二個根分母為7)

　

註: 若根為無理數，計數機會長時間顯示空白。
