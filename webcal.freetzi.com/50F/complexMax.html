二次函數極大/極小值及複數根

注意這個程式不單獨使用，必須配合內置一元二次方程(I)同時使用。

程式編寫日期: 2006年4月12日

程式長度: 14步

注意輸入程式前請先按 1 SHIFT Kin 1確保程式能順利輸入。
2 Kin ÷ 6 +/- Kin ÷ 2 Kin ÷ 6
Kout 1 Kin ÷ 2 Kin ÷ 6 F B
Kout 1 Kin ÷ 6 Kout 6 √ MODE .

　

例題1: 求函數 y = x2 + 7x + 12 的頂點。

按 1 FMLA 1 RUN 7 RUN 12 RUN P1 (顯示y坐標為 -0.25) RUN (顯示x坐標為 -3.5)

　

例題2: 解 x2 + 6x + 25 = 0

按 1 FMLA 再按 1 RUN 6 RUN 25 RUN (顯示-E-表示為複數根)

再按 AC P1 (顯示y坐標為 16) RUN (顯示x坐標及根實數部為 -3)

RUN (顯示根虛數部為 4)

所以方程的解為 x = -3 ± 4i
