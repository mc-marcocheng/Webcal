內部數值正常，相信是最後幾步輸入錯誤，或者我詳細寫出輸入程式的按法，要留意中間顯示的數值是否和我描述的完全一樣，希望這樣可以幫你找出問題所在，另外要特別留意最後幾步按法，好可能是最後的十個程式步出錯。

按 [MODE] [EXP] 這時顯示P1P2，再按 [SHIFT] [C] 清除所有程式記憶

再按 [P1] [FMLA] [ab/c] [1] 這時顯示A?及1

再按 [FMLA] [sin] [2] 這時顯示D?及2

再按 [FMLA] [cos] [2] 這時顯示E?及2

再按 [FMLA] [tan] [1] 這時顯示F?及1

再按 [Kout] [1] 這時顯示1

再按 [SHIFT] [Kout] [÷] [4]  [SHIFT] [Kout] [÷] [5]  [SHIFT] [Kout] [÷] [6] [2] [+/-] 這時顯示 - 2

再按   [SHIFT] [Kout] [÷] [4]  [SHIFT] [Kout] [÷] [5] [FMLA] [sin] 這時顯示 D?及- 1

再按 [x<sup>2</sup>] 這時顯示 1

再按 [+] [FMLA] [cos] 這時顯示 -1

再按 [x<sup>2</sup>] 這時顯示 1

再按 [-] 這時顯示 2

再按 [Kout] [6] 這時顯示 1

再按 [=] 這時顯示1

再按 [√] 這時顯示1

再按 [MODE] [.] 完成輸入

　

例題: 求圓的圓心及半徑: x2 + y2 – 4x + 6y – 12 = 0。 

按 P1  1 RUN (表示 x2 及 y2 的係數為1)

4 +/- RUN 6 RUN 12 +/- RUN (顯示2) RUN (顯示-3，即圓心為(2,-3))

RUN (顯示圓的半徑為5)
