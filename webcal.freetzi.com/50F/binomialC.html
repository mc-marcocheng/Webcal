註1: 二項式形式為 (axb + cxd)n，順序輸入a, b, c, d, n及要計算變數冪數。

註2: 若有方根形式，冪數以分數形式輸入，例如: 平方根以 1/2 輸入，若果是分數變數，請以負冪數輸入，若要計算常數項，請最後冪數輸入0)

　

例題1: 二項式(x2 - 3)9 展開後，求x6的係數。

 按 P1 再按 1 RUN 2 RUN 3 +/- RUN 0 RUN (0代表常數項)

9 RUN 6 RUN (顯示x6係數為61236)

　

例題2: 二項式(2x2 - 3/x )9 展開後，求常數項的係數。

 按 P1 再按 2 RUN 2 RUN 3 +/- RUN 1 +/- RUN (負冪數代表分數)

9 RUN 0 RUN (0冪數代表計算常數項，顯示常數項為489888)
