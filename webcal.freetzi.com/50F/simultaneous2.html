聯立二元一次方程式(II)

程式編寫日期: 2006年2月21日

這個程式較第 I版的簡短，因此可以保留較多程式空位，有機會可以輸入多一個程式，但要注意運用程式前要按 SHIFT KAC清除記憶。

程式長度: 19步

注意: 輸入程式之前，請先按 SHIFT KAC MODE 2 DATA
ENT 1 ÷ ÷ ENT =
xDyD ENT 1 = DATA MODE 0
1 Min Kout 3 x ≦ M MODE 2
SHIFT 7 HLT SHIFT 8 MODE 0 MODE .

註: 程式中的紅色的MODE 0可以不輸入，但程式終結時會在LR MODE，而程式的限制為方程的X係數不可以是零。

　

例題: 解聯立方程 :

 

先按 SHIFT KAC (必要，清除記憶)

按 P1 再按 1 RUN 1 RUN 7 RUN 1 RUN 1 +/- RUN 1 RUN (顯示4) RUN (顯示3)

因此解答為 x = 4 及 y = 3.
