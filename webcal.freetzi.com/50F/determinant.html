例題2: 計算 (i + 2j + 3k) x ( 4i + 5j + 6k)

按 1 P1  2 RUN 3 RUN 4 RUN 5 RUN 6 RUN

再按 Kout 1 (顯示-3) Kout 2 (顯示6) Kout 3 (顯示-3)

所以(i + 2j + 3k) x ( 4i + 5j + 6k) = -3i + 6j -3k

　

例題3: 計算 (i + 2j + 3k) x ( 4i + 5j + 6k)˙(2i - 3j + 7k)

按 1 P1  2 RUN 3 RUN 4 RUN 5 RUN 6 RUN  P1 2 RUN 3 +/- RUN 7 RUN (顯示答案為 - 45)

所以(i + 2j + 3k) x ( 4i + 5j + 6k)˙(2i - 3j + 7k) = - 45
