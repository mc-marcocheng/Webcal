平方根化簡(II)

　

程式可以化簡一個整數的平方根(將√(a/b)化簡為(c/d)√e，其中 a, b, c, d, e為正整數)，但要注意計算答案c/d可能未約簡，請自行約簡或再使內 置分數功能進行約簡。

程式第一部份(儲存於P1)長度: 7步

注意: 在輸入程式前請先按 MODE 1 DEC MODE 0
0 Kin 3 1 Kin 2 D 3
B 2 Kin × 4 MODE . 　 　

　

程式第二部份(儲存於P2)長度: 21步

注意: 在輸入程式前請先按 MODE 1 DEC MODE 0
1 Kin + 3 Kout 4 ÷ X←→K 3
Kin ÷ 3 Kout 3 √ MODE 1 MODE 0
Kin 1 x2 - MODE 1 MODE 0
Kin 3 = x > 0 A B
Kout 3 MODE . 　 　 　

　

例題1: 化簡 √18

按 P1 再按 18 RUN 再按 SHIFT P2 (顯示 3) RUN (顯示 1) RUN (顯示 2)

所以 √18 = (3/1)√2 = 3√2

　

例題2: 化簡 √(8/27)

按 P1 再按 8 RUN 27 RUN 再按 SHIFT P2 (顯示 6) RUN (顯示 27) RUN (顯示 6)

所以 √(8/27) = (6/27) √6 = (2/9) √6

注意分數部份答案(6/27)未約簡，請自行約簡或再使內置分數功能進行約簡。
