彈性碰撞

若已知兩物體質量及碰撞前的速度，程式可以計算經過彈性碰撞後兩物體的速度。

程式編寫日期: 2006年6月20日

程式長度: 20步

C 1 B 1 Kin ÷ 3 A 1 Kin 2
Kout 3 Kin × 1 Kout 2 Kin - 1 1
Kin + 3 ENT 0 Kin + 1 Kin + 1 Kin - 2
Kout 3 Kin ÷ 1 A Kin + 2 Kout 2
MODE . 　 　 　 　

　

例題: 物體A的質量及初速度分別為 5 kg及6 ms-1，物體B最初則靜止，質量為 10 kg，若物體A正面撞向物體B，碰揰為完全彈性碰撞，求A及B碰撞後的速度。

先輸入兩物的質量再輸入初速度

按 P1   5 RUN 10 RUN 6 RUN 0 RUN (顯示A的未速度為 - 2 ms-1)

EXE (顯示B的未速度為 4 ms-1)
