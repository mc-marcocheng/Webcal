餘弦定律(III)

注意: 這個程式要配合內置希羅公式及內置餘弦定律(I)使用。

若已知 三角形的三邊 (3 sides)， 程式會計算三角。

程式編寫日期: 2006年3月25日

程式長度: 20步

注意: 輸入程式前，請先按 SHIFT KAC 1 SHIFT Kin 2 SHIFT Kin 3
Kout 2 X←→K1 X←→K3 Kin 5 X←→K2
Kin × 5 x2 + Kout 2 x2
- Kout 3 x2 = ÷
2 ÷ Kout 5 = cos-1
MODE . 　 　 　 　

　

例題1: a=4, b=5, c=6，求三角形面積, ∠A, ∠B及∠C。

注意:要在DEG Mode執行

按 3 FMLA  再按 4 RUN 5 RUN 6 RUN (顯示面積為9.926)

再按 P1(顯示∠A=41.410o)

P1 (顯示∠B=55.771o)

P1 (顯示∠C=82.8192o)

　

例題2: a=4, b=5, ∠C=40o ，求c, ∠A及∠B。

注意:要在DEG Mode執行

按 2 FMLA  再按 4 RUN 5 RUN 40 RUN (顯示答案為c=3.2184)

SHIFT Kin 3 (必要) P1 (顯示∠A=53.024o)

P1 (顯示∠B=86.976o)

P1 (顯示∠C=40o)
