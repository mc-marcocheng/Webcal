平方根化簡(I)

程式編寫日期: 2006年2月24日

程式可以將一個正整數的平方根(√a)化簡為最簡式(b√c)。程式二較長，但操作較簡單。

程式一長度: 15步

注意: 在輸入程式前請先按 MODE 1 DEC MODE 0
1 Kin + 1 Kout 4 ÷ Kout 1
= √ Kin 2 - MODE 1
MODE 0 = x > 0 B Kout 1
MODE . 　 　 　 　

　

程式二長度: 22步

注意: 在輸入程式前請先按 MODE 1 DEC MODE 0
1 Kin + 1 Kout 4 ÷ Kout 1
= √ Kin 2 - MODE 1
MODE 0 = x > 0 B Kin 4
0 Min 1 X←→K3 Kin × 1
Kout 1 x ≦ M MODE . 　 　

　

例題: 化簡 √18

程式一的按法:

按 SHIFT KAC 18 SHIFT Kin 4  再按 P1 (顯示 3) RUN (顯示 2)

所以 √18 = 3√2

　

程式二的按法:

按 SHIFT KAC  P1 18 RUN (顯示 3) RUN (顯示 2)

所以 √18 = 3√2
