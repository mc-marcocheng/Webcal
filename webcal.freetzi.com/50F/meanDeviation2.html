例題: 計算以下一組數值75, 80, 80, 85, 85, 85, 85的平均數、標凖差及平均偏差。

計算前先清除記憶，按 SHIFT KAC 再按 MODE 3

75 DATA 80 DATA DATA 85 × 4 DATA

SHIFT x (顯示平均數為82.143) SHIFT xσn (顯示標凖差為3.642)

現在啟動程式再次輸入數據，注意若要輸入頻數，方法與內置統計不同，請先輸入頻數再按×及數值

再按 P1  75 RUN 80 RUN 80 RUN 4 × 85 RUN (顯示平均偏差為3.26531)

計算完結後按 MODE 0 返回正常計算模式
