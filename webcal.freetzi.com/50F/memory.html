記憶計算

編寫日期: 2006年4月6日

按鍵說明:

獨立記憶M

將數值儲存到獨立記憶:  按 SHIFT Min 

例如: 99 SHIFT Min

增加獨立記憶的數值:  按 M+

例如: 按 30 M+

減少獨立記憶的數值:  按 SHIFT M-

例如: 按 50 SHIFT M-

顯示及提取獨立記憶:  按 MR

註: M+及M-還有相當於等號=的完結功能。

常數記憶

將數值儲存到常數記憶: fx-50F 按 SHIFT Kin 再按記憶位置 (1至6)，fx-3900PV / fx-3800P / fx-3600PV 按 Kin 再按記憶位置 (1至6)。

例如: fx-50F: 按 99 SHIFT Kin 1

fx-3900PV / fx-3800P / fx-3600PV: 按 99 Kin 1

直接對常數記憶加減乘除: fx-50F 按 SHIFT Kin + 或 - 或 × 或 ÷ 再按記憶位置 (1至6)，fx-3900PV / fx-3800P / fx-3600PV 按 Kin + 或 - 或 × 或 ÷ 再按記憶位置 (1至6)。

例如: 按 99 SHIFT Kin + 1

顯示及提取常數記憶:  按 Kout 再按記憶位置 (1至6)

例如: 按 Kout 1
