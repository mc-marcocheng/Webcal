普通統計(I)

　

例題: 計算以下一組數值75, 80, 80, 85, 85, 85, 85的平均數及標凖差。

計算前先清除記憶，按 SHIFT KAC 再按 MODE 3

75 DATA 80 DATA DATA 85× 4 DATA

SHIFT x (顯示平均數為82.143) SHIFT xσn (顯示標凖差為3.642)

計算完結後，若果是fx-50F / fx-3800P 按 MODE 0 返回正常計算模式。若果是fx-3600PV按 MODE . 返回正常計算模式。

　

註: 計數機使用K1、K2及K3儲存統計數據的資料，請不要使用這些記憶儲存其它數值，否則會使統計計算出錯。
