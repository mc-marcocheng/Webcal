多項式除法(III)

這個程式計算一個N次多項式除以一個一次多項式的商及餘數。

程式編寫日期: 2006年2月9日

注意: 程式第二部份主要用作輸入除式及被除式的最高次數，若不想輸入這部份，亦可以自行將數值儲存於數字記憶中。

程式第一部份(P1)長度: 14步

注意: 輸入程式前，請先按 1 SHIFT Kin 1
Kout 1 Kin ÷ 5 Kout 5 Kin 4 E
Kout 2 Kin × 4 Kout 4 Kin - 5 1
Kin + 3 Kout 3 x ≦ M Kout 5 MODE .

程式第二部份(P2)長度: 5步 (可以不輸入)
KAC A B ENT Min
MODE . 　 　 　 　

　

例題: 計算 (2x4 + x3 + 3x2 + 4x + 5) ÷ (x + 2) 的商及餘數。

按 P2  再按 1 RUN 2 RUN (先輸入除式的係數)

4 RUN (輸入被除多項式的最高次數)

再按 P1  2 RUN (顯示商 x3 的係數為2) 1 RUN (顯示商 x2 的係數為 - 3)

3 RUN (顯示商 x 的係數為9) 4 RUN (顯示商常數項的係數為 - 14)

5 RUN (顯示餘數為33)

所以商為2x3 - 3x2 + 9x - 14，餘數為33。

　

另一按法 (不輸入程式第二部份)

按 SHIFT KAC 1 SHIFT Kin 1 2 SHIFT kin 2 (先輸入除式的係數)

4 SHIFT Min (輸入被除多項式的最高次數)

再按 P1  2 RUN (顯示商 x3 的係數為2) 1 RUN (顯示商 x2 的係數為 - 3)

3 RUN (顯示商 x 的係數為9) 4 RUN (顯示商常數項的係數為 - 14)

5 RUN (顯示餘數為33)

所以商為2x3 - 3x2 + 9x - 14，餘數為33。
