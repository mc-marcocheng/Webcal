一元二次方程(I)

例題1: 解 x2 - 7x + 12 = 0

按 1 FMLA 再按 1 RUN 7 +/- RUN 12 RUN (顯示第一個實數根為4) EXE (顯示第二個實數根為3)

∴ x = 4 或 x = 3

例題2: 解 x2 + 6x + 25 = 0

按 1 FMLA 再按 1 RUN 6 RUN 25 RUN (顯示-E-表示無實根)

　

註1: 程式執行完結後，按 Kout 6 顯示判別式的值，若有實數根，可按 Kout 4 及 Kout 5 分別顯示兩根的數值。

註2: 若果希望可以將有理數的根化為分數，可以使用 小數轉換分數(I) 或 小數轉換分數(II) 配合這個內置功能的程式。

註3: 若果希望可以同時計算二次函數因式，可以使用二次函數因式分解配合這個內置功能的程式。

註4: 若果希望可以同時計算二次函數的極大/極小點及複數根，可以使用下列的輔助程式:

二次函數極大/極小值 (Maximum/minimum value of quadratic function)

二次方程複數根 (Complex roots of quadratic equation)

二次函數極大/極小值及複數根 (Max./Min. value of quadratic function and complex roots)
