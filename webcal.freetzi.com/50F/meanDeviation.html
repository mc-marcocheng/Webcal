整合平均值及標準差

程式可以將多組已知平均值及標準差數據與計數機中的統計數據整合。輸入的標準差可以是樣本標準差或正常標準差。

程式編寫日期: 2006年3月26日

程式長度: 19步

注意: 輸入程式前，請先按 MODE 3
Min D Kin + 3 × E
× Kin + 2 Kout 5 + F
x2 × MR Kin - 4 Kout 4
= Kin + 1 MR x ≦ M MODE .

註: 若果為全新數據，請先按 SHIFT KAC清除統計記憶。

　

例題: 將下列的兩組數據整合為一組。

若使用的是正常標準差:

按  0 P1  (0表示使用正常標準差)

30 RUN 80 RUN 15 RUN 40 RUN 70 RUN 10 RUN 50 RUN 60 RUN 8 RUN

再按 SHIFT x (顯示平均值為68.33333333)

SHIFT xσn (顯示正常標準差為13.42158295)

　

若使用的是樣本標準差:

按 1 P1 (1表示使用樣本標準差)

30 RUN 80 RUN 15 RUN 40 RUN 70 RUN 10 RUN 50 RUN 60 RUN 8 RUN

再按 SHIFT x (顯示平均值為68.33333333)

SHIFT xσn-1 (顯示正常標準差為13.35603809)
