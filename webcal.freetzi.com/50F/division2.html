例題: 計算 (3x3 - 5x2 -  4x + 5) ÷ (x2 - x - 1)的商及餘數。

按 P1  3 RUN 5 +/- RUN 4 +/- RUN 5 RUN

1 RUN 1 +/- RUN 1 +/- RUN (顯示商的x係數為 3) RUN (顯示商的常數項為 -2)

RUN (顯示餘式的x係數為 -3) RUN (顯示餘式的常數項為3 )

所以商為3x - 2，餘式為 -3x + 3。
