二項分佈(I)

程式編寫日期: 2006年2月4日

程式長度: 16步 (注意: A是按 FMLA a b/c，B是按 FMLA  。'''，C是按 FMLA hyp)
A 2 nCr B 1 Kin - 1 ×
1 Kin 4 C 0 Kin - 4 xy
Kout 2 × Kout 4 xy Kout 1
M+ MODE . 　 　 　

注意: 程式除了計算答案外，亦會將答案累加在獨立記憶(M)中，若果要計算多個概率的總和，請先清除記憶M的數值(按 AC SHIFT Min)

　

例題1: 若X ~ Bin(9,0.5)，求P(X=4)。

按 P1  9 RUN 4 RUN . 5 RUN (顯示答案為0.24609)

　

例題2: 若X ~ Bin(9,0.5)，求P(X=4)，P(X=7)及P(X=4)+P(X=7)。

按 AC SHIFT Min (必要) P1  9 RUN 4 RUN . 5 RUN (顯示P(X=4)為0.24609)

P1  9 RUN 7 RUN RUN (顯示P(X=7)為0.07031)

MR (顯示P(X=4)+P(X=7)為0.31641)
