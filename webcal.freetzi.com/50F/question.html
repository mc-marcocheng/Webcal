解答:

1. 是不是fx-50F只可以輸入一個程式？為何當輸入第二個程式時總是出現-E-？

答: 計數機的兩個程式的總長度要小於29步，否則就無法輸入第二個程式，若果希望可以輸入兩個程式，一定要選擇較簡短的兩個程式。

　

2. 我只輸入一個程式，該程式的長度小於29步，為何總是在輸入中途出現-E-？

答: 可能是以下其中一個原因:

原因一: 輸入新程式前，有舊程式存在，佔用了程式的存儲存空間，所以只要先清除舊有的程式即可。

原因二: 輸入某些程式前，可能要預先將一些特別數值儲存數字記憶中，否則會出現運算的錯誤(-E-)，另外某些程式亦可能要預先設定好操作模式，否則同樣可能出現錯誤，請在輸入程式前細心閱讀程式說明。

　

3. 輸入程式有何步驟？一些特別的指令(例如: A)不知如何輸入？

答: 程式的輸入步驟如下:

步驟一: 按 MODE EXP (進入LRN模式)

步驟二: 按 P1 或 SHIFT P2 (選擇要輸入程式的位置)

步驟三: 輸入程式

步驟四: 按 MODE .  (離開LRN模式，返回RUN模式)

特別指令的按法如下:

4. 如何清除程式?

答: 先按 MODE EXP 進入學習模式，這時計數機顯示P1P2的細字，

     (i) 若果要將兩個程式同時清除，再按 SHIFT PCL即可。

     (ii) 若果只想清除其中一個程式，按 P1 (或者按 SHIFT P2) ，再按 SHIFT PCL即可。

     清除了程式，再按 MODE 0 返回正常計算模式。

　

5. 輸入了程式，為何程式會消失了？

答: 這可能是由於計數機的電池沒有電能，雖然可以利用光能繼續操作，但當沒有光的環境下，程式的記憶就不能保存，請更換新電池。

　

6. 編寫程式時，如何指定操作模式？

答: 在輸程式前(按 MODE EXP 前)，先設定好程式所用的操作模式，按 MODE 1 代表使用基數模式，或按 MODE 2代表使用線性回歸模式，或按 MODE 3代表使用普通統計模式，或按 MODE 0表示使用正常計算模式，接著再輸入程式。當執行程式式時，計數機會自動進入輸入程式時的模式，若果是基數模式，亦會自動進入輸入程式時的數制(二、八、十或十六進制)。注意在程式的編過程中，亦可以再按 MODE 0、1、2或3切換操作模式，但會佔用一步的程式空間。

　

7. 如何計算次方(例如: 210)及開多次方(例如: 5√32)的問題？

答: 使用計數機上 xy 及 x1/y 的功能(在×及÷的按鍵上)，

例如:計算210  按 2 SHIFT 10 = 計算5√32 按 32 SHIFT 5 =

　

8. 小數位及有效數字是如何設定及取消？

答:

小數設定 (FIX):            按 MODE 7 再按小數點後的位數 (0-9)

有效數字設定 (SCI):    按 MODE 8 再按有效數字的位數 (0-9，0代表取10個位)

取消小數或有效數字: 按 MODE 9

　

9. 計數機的操作模式如何設定?

答:

正常計模式: 按 MODE 0 (無顯示 BASE-N或LR或SD的細字)

基數計算模式: 按 MODE 1 (顯示BASE-N的細字)

線性回歸模式: 按 MODE 2 (顯示LR的細字)

普通統計模式: 按 MODE 3 (顯示SD的細字)

　

10. 三函數的度數及弧度是如何設定？

答: 使用度數按 MODE 4，計數機會顯示D的細字，使用弧度數按 MODE 5，計數機會顯示R的細字。

　

11. 如何將計算的結果儲存在計數機中？

答: 將數值儲存到獨立記憶:  按 SHIFT Min，

將數值儲存到常數記憶: 按 SHIFT Kin 再按記憶位置 (1至6)。

以上的操作不影響計算的運算，因此可以在計算的中途進行亦可。

　

12. 為何計算平均數及標準差總是很容易出錯？

答: 出現錯誤的原因可能很多，可以注意下列幾點，看看情況可否改善。

(1) 在輸入新數據前，必定要先清除原先的統計記憶 (按 SHIFT KAC)，否則計算很可能出錯。

(2) 在計算普通統計問題時，不應使用常在記憶(K1、K2及K3)儲存數值，因為這些記憶用作統計功能的運算，若有需要儲存數值，請盡可能使用獨立記憶M。

(3) 由於可能要輸入不少的統計數據，輸入時要較為小心，輸入完成後，按 Kout 3顯示輸入數據的總數目，核對真實數據的數目是否相同，若果不相同表示輸入錯誤。

　

13. 如何輸入有10的指數部分的數值(例如: 2.1×1011)？

答: 使用 EXP 這個按鍵，例如: 2.1×1011 按 2.1 EXP 11

　

14. 為何無法使用內置程式庫的程式？

答: 使用內置程式庫的程式必須在 comp 模式中執行，若果不是在 comp 模式，請按 MODE 0 進入 comp 模式。另外每一個內置程式都有一個編號，使用程式時，請先按出編號再按 FMLA 進入特定的程式，若果沒有按編號，直接按 FMLA ，將會使用上一次使用程式庫的程式。

　

15. 輸入的數值發現錯誤，有沒有方法進行修改?

答: 若果剛剛輸入數值，有兩個方法可以修改。第一個方法是按 C 完全清整個數值，然後重新再輸數值即可。第二個方法是按向右鍵(→)，每按一次向右鍵可以刪除原先數值最後的一個數位。

　

16. 甚麼是鎖機？有沒有解決的方法？

答: 鎖機是計數機的一種設計上錯誤，當計數機被鎖，再按計數機上任何一個按鍵都沒有反應。這個特點亦成為一種惡作劇的方法。解決的方法如下:

(1) 等待六分鐘，等待的過程要注意千萬不可按任何按鍵，只要計數機沒有任何運算，計數機會自動關機，這時就可以解鎖，計數機會回復正常。

(2) 盡量保管好自己的計數機，不要借計數機給別人使用。

(3) 若果問題很嚴重，可以將計數機中的電池取去，因為一般室內的光線(例如: 課室)足夠fx-50F以光能正常運作，若果計數機被鎖，只要遮蓋太陽能版，因能量不足，計數機會自動關機，這時當然亦解了鎖，但要注意若果使用這個方法，就不能保存輸入的計數機程式，不過計數機的程式功能亦依然可以留作函數計算及迭代法計算(iteration method)。

註: 在重要的考試(公開試時)，為了確保有足夠能量使計數機正常運作，最好都是把電池放回計數機中。

　

17. 傳聞 fx-50F有關機方法，是否屬實？

答: fx-50F並無立即關機的方法，要等計數機啟自動關機功能才可以。至於那個用分數除以0的方法，雖然可以令計數機不顯示任何資料，但計數機並沒有關機，相反計數機會一直繼續運行，並且無法啓動自動關機功能，因此會一直損耗電池的能量。

　

18. 為何計數機的數字有時變得暗淡，速度亦變慢，但有時又正常？

答: 好可能是電池的能量已經或將近用盡，雖然fx-50F可以在較暗的環境下單靠光能運作，不過在能量不足的情況下，是可以令計數機的數字變淡及減慢運算速度，當然在光線較足夠的環境下，自然又可回復正常。

　

19. fx-50F使用何類電池，如何更換電池？

答: fx-50F使用的電池為 SR927。

更換電池的步驟如下:

(1) 將計數機的背殼固定的6個螺絲擰開，然後打開背殼。

(2) 將舊電池取出。

(3) 把新電池的正極 + 向上地裝入機內。

(4) 裝回背殼，並用6個螺絲將其固定。

　

20. fx-50F可否單靠光能運作？

答: fx-50F使用特別的 C-power 系統，因此在一般室內的燈光下，是可以正常運作，不過若果在光線較暗的情況下，計數機的運算速度可能會減慢甚至不能運作。

　

21. 如何分辨水貨及行貨計數機？

答: 購買計數機要留意有沒有保用証，保用証是不是由Casio授權那個地方的合法代理(可參看Casio的網址 http://world.casio.com/asia/support/distributors_cal.html )，若果是水貨就沒有保養服務。
