基數運算

　

例題1: 將2610轉為二進制、八進制及十六進制。

先進入基數模式，fx-50F / fx-3800P 按 MODE 1 ，fx-3600PV 按 MODE 0

fx-50F / fx-3600PV按法:

按 DEC 26 SHIFT BIN (顯示11010 b，即是110102)

再按 SHIFT OCT (顯示32 o，即是328)

再按 HEX (顯示1A H，即是1A16)

fx-3800P 按法:

按 DEC 26 BIN (顯示11010 b，即是110102)

再按 OCT (顯示32 o，即是328)

再按 HEX (顯示1A H，即是1A16)

計算完結後， 若果是fx-50F / fx-3800P按 MODE 0 返回正常計算模式，若果是fx-3600PV 按 MODE . 返回正常計算模式。

 

例題2: 計算 1FA16 + 5078，最後答案以十六進制表示。

先進入基數模式，fx-50F / fx-3800P 按 MODE 1 ，fx-3600PV 按 MODE 0

fx-50F / fx-3600PV按法:

再按 HEX 1FA + SHIFT OCT 507 = HEX (顯示341 H，即是34116)

fx-3800PV按法:

再按 HEX 1FA + OCT 507 = HEX (顯示341 H，即是34116)

計算完結後， 若果是fx-50F / fx-3800P按 MODE 0 返回正常計算模式，若果是fx-3600PV 按 MODE . 返回正常計算模式。

 

例題3: 計算 10112 AND 11002

先進入基數模式，fx-50F 按 MODE 1 ，fx-3600PV 按 MODE 0

fx-50F 再按 SHIFT BIN 1011 AND 1100 = (顯示1000b，即是10002)

fx-3600PV 再按 BIN 1011 AND 1100 = (顯示1000b，即是10002)

　

計算完結後， 若果是fx-50F 按 MODE 0 返回正常計算模式，若果是fx-3600PV 按 MODE . 返回正常計算模式。
