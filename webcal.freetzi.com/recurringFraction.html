循環小數轉換分數

程式編寫日期: 2006年3月4日

0 A + B + 1 a b/c 2
÷ ( 10x C - 1 ) ×
2 D ) ÷ 10x A STO F1 或 F2

例題1: 將 0.2333333.....化為分數

按 RCL F1 ALGB 1 = (輸入小數點後出現循環前的位數)

2 = (輸入小數點後循環前出現的數字)

1 = (輸入最短的循環長度(週期))

3 = (輸入最短循環的數字，顯示答案為 7/30)

　

例題2: 將 0.45454545......化為分數

按 RCL F1 ALGB 0 = (輸入小數點後出現循環前的位數)

0 = (輸入小數點後循環前出現的數字，0代表沒有)

2 = (輸入最短的循環長度(週期))

45 = (輸入最短循環的數字，顯示答案為 5/11)

　

例題3: 將 2.45454545......化為分數

按 RCL F1 ALGB 0 = (輸入小數點後出現循環前的位數)

2 = (輸入小數點後循環前出現的數字)

2 = (輸入最短的循環長度(週期))

45 = (輸入最短循環的數字，顯示答案為 2/5/11)

再按 2ndF d/c (顯示假分數為 27/11)
